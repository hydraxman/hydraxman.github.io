前言

2026年2月23日，GitHub Trending 榜单呈现出几个鲜明趋势：AI Agent 无处不在——从安全渗透到编码辅助再到云端部署，Agent 正在重塑开发者工作流；Prompt 透明化继续升温，11万星的 system-prompts 仓库说明开发者对 AI 黑箱的好奇心只增不减；开发者工具方面，代码知识图谱和等距架构图工具各自找到了独特切入点。

1. vxcontrol/pentagi — AI 全自动渗透测试

PentAGI（Penetration testing AGI）是一个用 Go 编写的全自动 AI 渗透测试系统，今日狂揽 +1,579 stars，总星数突破 7.2K。

为什么火？

安全测试长期依赖人工经验，PentAGI 把整个流程交给 AI Agent：自动选择攻击路径、在沙箱 Docker 环境中执行、生成漏洞报告。内置 20+ 专业安全工具（nmap、metasploit、sqlmap 等），支持多 LLM 后端（OpenAI、Anthropic、Ollama、Bedrock 等）。

技术亮点

  • 知识图谱记忆：基于 Neo4j + Graphiti 的语义关系追踪，让 AI 能复用历史攻击经验
  • 专家委派系统：主 Agent 可将子任务委派给研发、基础设施等专项 Agent
  • 完全隔离执行：所有操作在沙箱容器内完成，保障安全

适用场景

安全团队的自动化渗透预检、红队演练辅助。注意：这是辅助工具，不能替代专业安全审计。

⭐ 7,289 | 📈 +1,579 today | Language: Go

2. x1xhlol/system-prompts-and-models-of-ai-tools — AI 工具 System Prompt 大全

这个仓库收集了 30+ 款主流 AI 工具的 System Prompt，包括 Cursor、Claude Code、Devin、Manus、Windsurf 等，今日再涨 +1,134 stars，总星数冲破 11.8 万。

为什么火？

开发者对 AI 编码工具的内部机制充满好奇。这个仓库提供了超过 3 万行的 prompt 内容，让你看到这些工具”思考”的方式——从工具调用策略到安全约束，从代码生成模板到错误处理逻辑。

价值所在

  • 学习顶级 AI 产品的 prompt 工程实践
  • 理解不同工具的能力边界和设计哲学
  • 为自己构建 AI Agent 提供参考模板

⭐ 118,196 | 📈 +1,134 today

3. anthropics/claude-code — Anthropic 官方终端编码工具

Claude Code 是 Anthropic 推出的终端 AI 编码助手，理解你的代码库，通过自然语言执行日常开发任务。今日 +503 stars，总星数近 6.9 万。

技术亮点

  • 终端原生：不需要 IDE 插件，直接在终端中运行
  • 深度代码理解：自动索引项目结构，理解上下文
  • Git 工作流集成：代码审查、提交、分支管理一体化
  • 插件生态：支持自定义命令和 Agent 扩展

适用场景

日常编码提效——代码解释、重构、Bug 修复、Git 操作。特别适合习惯终端工作流的开发者。

⭐ 69,081 | 📈 +503 today | Language: Shell

4. huggingface/skills — HuggingFace 的 AI Agent 技能生态

HuggingFace 推出的 Skills 仓库，为各大编码 Agent（Claude Code、Codex、Gemini CLI、Cursor）提供标准化的 AI/ML 任务技能包。今日 +364 stars。

为什么火？

AI Agent 的能力取决于它能调用的”技能”。HuggingFace 将模型训练、数据集管理、评估等 ML 任务封装成标准化技能包，让任何 AI 编码工具都能直接使用。这是 Agent 技能标准化的重要一步。

可用技能

包括 HF CLI 操作、数据集创建管理、模型评估、计算任务调度、模型微调（SFT/DPO/GRPO）、论文发布等 7 个技能。

适用场景

ML 工程师用 AI Agent 自动化 HuggingFace 工作流——从数据准备到训练到部署，全链路 Agent 化。

⭐ 2,930 | 📈 +364 today | Language: Python

5. abhigyanpatwari/GitNexus — 零服务器代码知识图谱引擎

GitNexus 将代码库索引为知识图谱——追踪每个依赖、调用链、执行流——然后通过 MCP 协议暴露给 AI Agent。今日 +424 stars。

为什么火？

AI 编码工具最大的痛点是”上下文盲区”：不了解代码间的依赖关系就会写出破坏性修改。GitNexus 用知识图谱解决这个问题，让 AI Agent 看到完整的架构视图。

技术亮点

  • 双模式：CLI + MCP 用于日常开发；Web UI 用于快速探索（浏览器内运行，无需服务器）
  • 深度集成：Claude Code 支持 PreToolUse Hook，自动为 grep/glob 调用注入图谱上下文
  • KuzuDB 存储：本地图数据库，快速持久化

一句话

给你的 AI 编码助手装上”建筑蓝图”，让它不再盲目修改代码。

⭐ 1,601 | 📈 +424 today | Language: TypeScript

6. stan-smith/FossFLOW — 开源等距架构图工具

FossFLOW 是一款基于 React 的 PWA 应用，用于绘制精美的等距（Isometric）基础设施架构图。今日 +340 stars，总星数 1.8 万。

为什么火？

架构图工具市场中，等距视图一直是个空白。FossFLOW 填补了这个缺口——完全在浏览器中运行，支持离线使用，Docker 一键部署。对比 draw.io 和 Excalidraw，它的独特价值在于开箱即用的等距渲染。

适用场景

基础设施文档、技术博客配图、架构设计演示。特别适合需要”高颜值”架构图的技术写作场景。

⭐ 18,252 | 📈 +340 today | Language: TypeScript

7. cloudflare/agents — Cloudflare 的 AI Agent 部署框架

Cloudflare 基于 Durable Objects 打造的 AI Agent 框架，提供持久化状态、实时通信、调度、MCP 支持等能力。今日 +257 stars。

技术亮点

  • 按需休眠：Agent 空闲时零成本，唤醒即用
  • 百万级并发：每用户/每会话可独立运行一个 Agent
  • @callable 装饰器：类型安全的 RPC 调用，前端像调本地函数一样调用 Agent 方法
  • 内置 MCP：既可作为 MCP Server 也可作为 MCP Client

适用场景

需要持久化、有状态 AI Agent 的 SaaS 应用。Cloudflare 的边缘网络 + Durable Objects 提供了独特的部署优势。

⭐ 3,807 | 📈 +257 today | Language: TypeScript

8. Stremio/stremio-web — 开源流媒体中心

Stremio 是一个现代流媒体中心，通过插件系统聚合视频内容的发现、观看和组织。今日 +247 stars，总星数近 1 万。

为什么火？

在流媒体碎片化的时代，Stremio 提供了”一站式”体验——通过社区插件聚合多个视频源。开源、自托管、跨平台（Web + Docker），隐私友好。

适用场景

个人媒体中心搭建，适合追求自由和隐私的用户。

⭐ 9,763 | 📈 +247 today | Language: JavaScript

9. OpenBB-finance/OpenBB — 开源金融数据平台

OpenBB 是面向分析师、量化交易员和 AI Agent 的金融数据平台，今日 +188 stars，总星数突破 6.1 万。

技术亮点

  • Connect Once, Consume Everywhere：Python / Excel / REST API / MCP 多端消费
  • AI Agent 集成：提供专门的 Agent 接入层，金融数据直接接入 AI 工作流
  • 丰富数据源：股票、期权、加密货币、宏观经济数据等

适用场景

量化分析师的数据获取层、金融 AI Agent 的数据基座、个人投资研究工具。

⭐ 61,080 | 📈 +188 today | Language: Python

总结

今日榜单的关键词是 Agent 基础设施。从 HuggingFace Skills 的技能标准化，到 Cloudflare Agents 的部署框架，再到 GitNexus 的代码感知、PentAGI 的安全自动化——AI Agent 正在从概念走向工程化。

值得关注的趋势：

  1. Agent 技能标准化：HuggingFace Skills 的跨平台兼容方案值得借鉴
  2. 代码感知能力：GitNexus 用知识图谱解决 AI 编码的上下文盲区
  3. 安全领域 AI 化：PentAGI 的爆发式增长说明安全自动化需求旺盛
  4. 金融 + AI：OpenBB 的 MCP 集成让金融数据直接接入 Agent 生态
