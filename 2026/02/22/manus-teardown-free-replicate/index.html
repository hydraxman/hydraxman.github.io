Manus值$20亿，但你可以免费复刻它的核心逻辑

2025年12月30日，Meta（Facebook母公司）宣布以超过20亿美元收购AI Agent创业公司 Manus。

这是一家全中国人创立的公司，从产品发布到被收购仅用了9个月，创造了140倍的估值增长。

Reddit上已经有开发者逆向工程了Manus的核心工作流，并把它变成了开源项目。今天我们就来拆解：Manus到底做对了什么？普通人能不能用免费工具复刻它的核心逻辑？

Manus 的疯狂时间线

时间 事件
2025年3月 Manus正式发布，邀请码被炒到5万元
2025年5月 获得Benchmark 7500万美元B轮融资
2025年7月 撤出中国市场，团队迁至新加坡
2025年8月 ARR（年经常性收入）突破1亿美元
2025年12月 Meta以20亿+美元收购，十天内敲定

真格基金合伙人评价：”不靠关系，不比资历，在全球舞台上光明正大同台竞技，做到了我们上一代创业者做不了甚至不敢想的事情。”

Manus 的核心技术：不是模型，是”编排”

关键认知：Manus没有自己的大模型。它的价值在于Agent编排层（Agentic Harness）。

什么意思？打个比方：

  • 大模型 = 一个超级聪明的大脑
  • Manus = 给这个大脑装上了手脚、眼睛和工具箱

Manus的核心能力是让AI 自主完成复杂任务：不是你问一句它答一句，而是你给它一个目标，它自己规划步骤、调用工具、执行任务、检查结果。

Manus 的4层核心架构

根据多方技术分析和开源复现，Manus的架构可以拆解为4层：

第1层：任务规划（Planning）

用户给出一个复杂任务后，Manus会先拆解成子任务，生成一个Markdown格式的执行计划。

这是Manus最值钱的能力之一——把模糊的需求变成结构化的执行方案。Reddit上有开发者专门把这个”Planning with Files”模式做成了开源工具。

第2层：工具调用（Tool Use）

Manus能调用各种外部工具：

  • 🌐 浏览器：自动打开网页、搜索、抓取信息
  • 💻 代码执行：写代码并运行
  • 📊 数据分析：处理Excel、CSV等数据
  • 📁 文件操作：创建、编辑、管理文件

第3层：多Agent协作（Multi-Agent）

复杂任务不是一个Agent能搞定的。Manus用了多Agent协作模式：

  • 规划Agent：负责拆解任务、分配工作
  • 搜索Agent：负责信息检索和数据收集
  • 编码Agent：负责写代码、做计算
  • 执行Agent：负责具体操作（发邮件、调API等）
  • 审查Agent：检查结果质量，决定是否需要重做

这些Agent之间通过消息传递协调工作，就像一个高效的团队。

第4层：记忆与学习（Memory）

Manus会记住之前的任务和结果，下次遇到类似任务时能更快更好地完成。这让它越用越聪明。

用免费工具复刻”迷你版Manus”

好消息是：Manus的核心逻辑并不神秘，你完全可以用开源工具搭建一个类似的系统。

方案一：n8n + DeepSeek（推荐新手）

n8n 是最适合复刻Manus工作流的工具，因为它天然支持：

  • 可视化工作流编排（对应Manus的规划层）
  • 1000+外部服务集成（对应工具调用层）
  • AI Agent节点（原生支持LangChain Agent）
  • 条件分支和循环（对应多Agent协作）

方案二：Dify + OpenClaw（适合进阶用户）

如果你想要更接近Manus的”多Agent协作”能力：

  1. Dify：搭建各个专业Agent（搜索Agent、分析Agent、写作Agent）
  2. OpenClaw：作为编排层，协调多个Agent的工作流程
  3. 部署方式：全部Docker自部署，成本为零

这个方案更复杂，但能力也更强——可以处理真正的多步骤、跨系统的复杂任务。

方案三：Claude Code / Codex CLI（程序员专属）

如果你是程序员，最直接的方式是用终端AI Agent：

  • Claude Code（Opus 4.6驱动）：能自主读写代码、执行命令、管理项目
  • Codex CLI（Codex 5.3驱动）：OpenAI的终端编程Agent

Reddit上那个”逆向工程Manus”的帖子，就是用Claude Code实现的。核心思路是用Markdown文件做持久化规划——让Agent把任务计划写进文件，每完成一步就更新文件状态，跟Manus的Planning层如出一辙。

Manus被收购告诉我们什么？

1. 编排能力 > 模型能力

Manus没有自研大模型，但被Meta以20亿收购。这说明：会用AI比会造AI更值钱。对普通人来说，这是好消息——你不需要懂深度学习，只需要学会编排AI工作流。

2. 速度就是壁垒

从发布到被收购9个月，ARR 8个月破1亿美元。在AI Agent领域，先做出来比做得完美更重要。

3. 中国团队的全球化路径

Manus证明了中国技术团队完全有能力在全球舞台上竞争。对于想在AI领域创业的人来说，这是一个极大的信心提振。

4. Agent赛道刚刚开始

CB Insights最新报告《AI Agent圣经》指出，2026年AI Agent市场的6大趋势之一就是”行业专属Agent加速落地”。这意味着：垂直领域的Agent机会远未被填满。
