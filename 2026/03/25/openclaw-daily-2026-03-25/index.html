版本更新

v2026.3.22-beta.1（预发布）

这是一个重量级 Beta，包含多项 Breaking Changes，升级前务必仔细阅读：

🔴 Breaking Changes：

变更 影响 迁移建议
插件安装优先走 ClawHub openclaw plugins install <pkg> 现在优先查 ClawHub，npm 降为后备 无需改动，更快更稳
Chrome 扩展中继移除 删除 driver: "extension" 和 browser.relayBindHost 运行 openclaw doctor --fix 自动迁移
插件 SDK 统一为 openclaw/plugin-sdk/ 旧 openclaw/extension-api 彻底移除 按 SDK 迁移指南 迁移
消息发现改用 describeMessageTool() 旧 listActions/getCapabilities 删除 插件作者需更新适配器
Matrix 插件全新重写 基于官方 matrix-js-sdk 参考 Matrix 迁移指南
环境变量清理 CLAWDBOT_* 和 MOLTBOT_* 前缀彻底移除 统一使用 OPENCLAW_*
状态目录清理 ~/.moltbot 自动检测取消 手动迁移到 ~/.openclaw

新功能亮点：

  • 图片生成统一为 image_generate 工具，推荐使用 agents.defaults.imageGenerationModel 配置
  • 全新插件 SDK 公开接口，第三方插件开发更规范

v2026.3.23（稳定版修复）

紧跟 Beta 之后，稳定版发布了一批重要修复：

  • Qwen/阿里云模型: 新增标准 DashScope 按量付费端点，中国和全球 API Key 均可直接使用
  • UI 大改: Knot 主题升级为黑红配色，WCAG 2.1 AA 无障碍达标，新增配置图标
  • CSP 安全加固: 内联脚本改用 SHA-256 哈希白名单
  • ClawHub 兼容性: 修复 >=2026.3.22 版本检查导致的安装失败
  • 浏览器 CDP: 修复 Linux 无头环境下二次启动的重连问题
  • 认证 token: 修复 OpenAI token 保存后被旧值覆盖的经典 Bug
  • web_search: 修复搜索提供商配置不生效的问题

NVIDIA NemoClaw 重磅发布

GTC 2026 上，NVIDIA 正式宣布 NemoClaw —— 为 OpenClaw 量身打造的企业级 AI Agent 安全栈。

核心卖点：

  • 🚀 一条命令安装：Nemotron 模型 + OpenShell 沙箱运行时一键部署
  • 🔒 隐私与安全：策略驱动的网络/隐私护栏，Agent 既能干活又不乱来
  • 💻 全平台支持：从 RTX PC 到 DGX Station/Spark，云端到本地都能跑
  • 🤖 开放模型优先：支持本地 Nemotron 推理，也可通过隐私路由使用云端模型

Jensen Huang 原话：

“Mac 和 Windows 是个人电脑的操作系统，OpenClaw 是个人 AI 的操作系统。”

Peter Steinberger（OpenClaw 创始人）：

“我们正在构建让每个人都能创建强大、安全 AI 助手的工具和护栏。”

这标志着 OpenClaw 从个人玩具走向企业级应用的重要一步。想尝试的话，关注 nvidia.com/nemoclaw。
