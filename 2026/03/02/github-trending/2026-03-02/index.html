前言

今天的 GitHub Trending 几乎被 AI Agent 和 智能体编排 主题占领——从多智能体协作到沙箱执行环境，从科学研究技能包到手机GUI自动化，AI 正在以前所未有的速度渗透到开发者的每一个工作流中。而今天的最大黑马，是一个用 WiFi 信号透视人体姿态 的 Rust 项目，一天斩获 4500+ star。

1. ruvnet/wifi-densepose 🏆 今日之星

项目简介

用普通 WiFi 信号实现人体姿态估计、生命体征监测和存在检测——不需要摄像头，不需要穿戴设备，甚至可以穿墙。

为什么火？

这个项目击中了隐私与感知的完美交汇点。摄像头监控一直饱受争议，而 WiFi DensePose 用无处不在的无线信号替代了视觉方案：可以检测呼吸频率（6-30次/分）、心率（40-120bpm），甚至穿透墙壁最远5米。灾难救援场景下寻找被困者，这简直是革命性的。

技术亮点

用 Rust 编写，核心管线达到 54K fps 处理速度。通过分析 WiFi CSI（信道状态信息）的子载波振幅和相位变化，结合注意力网络和图算法进行信号处理。支持 ESP32-S3 硬件（成本仅约$54），也提供 Docker 一键部署。自学习架构无需标注数据，训练一次可跨环境部署。

适用场景

智能家居隐私安全监测、老人跌倒检测、灾难搜救、睡眠监测。对隐私敏感的场景极其友好。

⭐ Stars: 18,727 | 📈 Today: +4,539 | Language: Rust

2. microsoft/markitdown

项目简介

微软出品的文件转 Markdown 工具，支持 Office 文档、PDF 等多种格式到 Markdown 的转换。

为什么火？

在 LLM 时代，几乎所有工作流都需要把非结构化文档转成纯文本或 Markdown 来喂给模型。markitdown 直击这个痛点，而且有微软背书，质量和维护都有保障。近 9 万 star 说明了一切。

技术亮点

Python 实现，支持 docx/pptx/xlsx/pdf 等主流格式，可作为 CLI 工具或 Python 库使用。转换保真度高，对表格、列表等复杂结构处理出色。

适用场景

RAG 数据预处理、文档知识库构建、LLM 输入预处理、办公自动化。

⭐ Stars: 89,149 | 📈 Today: +805 | Language: Python

3. moeru-ai/airi

项目简介

开源的 AI 虚拟伴侣项目，灵感来自 Neuro-sama，支持实时语音聊天、Live2D 形象、甚至能玩 Minecraft 和 Factorio。

为什么火？

这是目前最接近”拥有自己的 Neuro-sama”的开源方案。Neuro-sama 火遍全网但不开源，AIRI 填补了这个空白。支持 Web/macOS/Windows 全平台，可自托管，你拥有自己的数字生命。

技术亮点

TypeScript 全栈，大量使用 WebGPU、WebAudio、WebAssembly 等现代 Web 技术。桌面版支持原生 CUDA 和 Metal 加速。项目拆分为多个子项目（RAG、记忆系统、嵌入数据库等），架构清晰。

适用场景

AI VTuber 开发者、虚拟伴侣爱好者、数字人研究、直播互动。

⭐ Stars: 20,625 | 📈 Today: +736 | Language: TypeScript

4. ruvnet/ruflo

项目简介

面向 Claude 的企业级多智能体编排平台，支持 60+ 专用 Agent 的协调调度。

为什么火？

Ruflo（前身 claude-flow）是目前针对 Claude Code 最成熟的多智能体框架之一。v3.5 版本经历了 5800+ 次提交，支持多种拓扑结构（网状/层级/环形/星型）、多种共识协议（Raft/BFT/Gossip/CRDT），以及 WASM 驱动的策略引擎。

技术亮点

TypeScript + Rust WASM 内核，支持 Q-Learning 路由、MoE 8专家系统、42+ 技能、自学习/自优化循环。支持 Claude/GPT/Gemini/Ollama 多种模型后端。

适用场景

企业级 AI 工作流编排、复杂软件工程任务分解、多 Agent 协作开发。

⭐ Stars: 17,499 | 📈 Today: +766 | Language: TypeScript

5. bytedance/deer-flow

项目简介

字节跳动开源的 SuperAgent 框架 DeerFlow 2.0，从零重写，支持子 Agent、记忆、沙箱和可扩展技能。

为什么火？

DeerFlow 2.0 在 2 月 28 日发布后直接冲上 GitHub Trending 第一名。从”深度研究工具”进化为”通用超级 Agent 框架”，能处理从几分钟到几小时的不同复杂度任务。基于 LangChain/LangGraph 构建，生态融合度高。

技术亮点

Python + TypeScript 双栈，支持沙箱模式、MCP Server、长期记忆、上下文工程。Docker 一键部署，支持 GPT-4、Claude 等多模型。2.0 与 1.x 完全无代码共享，是彻底的架构重设计。

适用场景

深度研究、自动化代码生成、复杂任务编排、企业知识工作流。

⭐ Stars: 23,156 | 📈 Today: +355 | Language: Python

6. alibaba/OpenSandbox

项目简介

阿里巴巴开源的通用 AI 沙箱平台，提供多语言 SDK 和统一 API，支持 Docker/K8s 运行时。

为什么火？

AI Agent 需要安全的代码执行环境，OpenSandbox 正好解决这个基础设施问题。一天 1179 star 的爆发说明社区对 Agent 沙箱的需求极其旺盛。支持 Coding Agent、GUI Agent、Agent 评估、RL 训练等多种场景。

技术亮点

Python/Java/TypeScript/C#/Go 多语言 SDK，统一的沙箱协议定义生命周期和执行 API。内置命令行、文件系统、代码解释器，支持 Chrome/Playwright 浏览器自动化和 VNC 桌面环境。网络策略支持 Ingress Gateway 和 per-sandbox Egress 控制。

适用场景

AI Agent 安全执行环境、在线代码编辑器后端、自动化测试、模型评估平台。

⭐ Stars: 3,588 | 📈 Today: +1,179 | Language: Python

7. Shubhamsaboo/awesome-llm-apps

项目简介

LLM 应用合集，涵盖 RAG、AI Agent、多智能体团队、MCP、语音 Agent 等方向的实战项目。

为什么火？

接近 10 万 star 的超级合集项目，堪称 LLM 应用的”百科全书”。覆盖 OpenAI、Anthropic、Google、xAI 以及开源模型（Qwen、Llama），每个项目都可以本地运行。对于想快速入门或寻找灵感的开发者来说，这是金矿。

适用场景

LLM 应用学习、项目灵感、技术选型参考。

⭐ Stars: 98,839 | 📈 Today: +471 | Language: Python

8. K-Dense-AI/claude-scientific-skills

项目简介

148+ 科学研究技能包，覆盖生物信息学、化学信息学、临床研究、材料科学等领域，支持 Cursor/Claude Code/Codex。

为什么火？

把 AI 编码 Agent 变成”桌面 AI 科学家”——这个定位太精准了。连接 250+ 科学数据库（PubMed、ChEMBL、UniProt 等），从基因组分析到药物发现，从蛋白质工程到金融 SEC 研究，一个技能包搞定。

技术亮点

基于开放的 Agent Skills 标准，每个技能包含完整文档、代码示例和最佳实践。不仅限于预定义技能，Agent 可以使用任何 Python 包，技能只是提供了优化的文档路径。

适用场景

科研工作者、药物研发、临床数据分析、学术写作、生物信息学。

⭐ Stars: 10,468 | 📈 Today: +189 | Language: Python

9. basecamp/omarchy

项目简介

DHH（Ruby on Rails 创始人）打造的美观、现代、有态度的 Linux 发行版。

为什么火？

DHH 的个人品牌效应 + Linux 社区对”开箱即用的美观桌面”的长期渴求。2 万+ star 主要靠的是 DHH 的号召力和”Opinionated”（有态度的）设计哲学——不是给你 100 个选项，而是直接告诉你最佳实践。

技术亮点

Shell 脚本为主，专注于桌面体验的开箱即用配置。

适用场景

想要美观 Linux 桌面的开发者、DHH 粉丝、厌倦了折腾 Linux 配置的人。

⭐ Stars: 20,778 | 📈 Today: +59 | Language: Shell

10. X-PLUG/MobileAgent

项目简介

阿里通义实验室的 GUI Agent 系列，最新 v3.5 版本搭载 GUI-Owl 1.5 多平台原生 GUI Agent 基础模型。

为什么火？

GUI-Owl 1.5 刚发布（2026.2.14），支持桌面/移动/浏览器三端自动化，在 20+ GUI 基准测试上达到 SOTA。提供从 2B 到 235B 多种规模的模型，还有 Instruct 和 Thinking 两种模式。阿里百炼平台提供了限时免费 API。

技术亮点

基于 Qwen3-VL 构建，支持端到端操作、视觉定位、工具/MCP 调用和长期记忆。Mobile-Agent-v3 框架提供规划、进度管理、反思和记忆能力。在 OSWorld 和 AndroidWorld 基准上都有评估代码和检查点。

适用场景

移动端自动化测试、RPA、跨平台 GUI 自动化、手机 AI 助手。

⭐ Stars: 7,781 | 📈 Today: +190 | Language: Python

总结

今天的 Trending 呈现出三大趋势：

  1. Agent 基础设施爆发：从 OpenSandbox（沙箱）到 Ruflo/DeerFlow（编排），AI Agent 的”水电煤”正在迅速完善。
  2. 跨模态感知突破：WiFi DensePose 用无线信号替代摄像头，MobileAgent 用视觉模型操控 GUI，AI 正在学会”看”和”感知”真实世界。
  3. 科研工具民主化：Claude Scientific Skills 让每个开发者都能拥有一个”桌面 AI 科学家”，门槛前所未有地低。

Agent 的时代正在加速到来，而今天的这些项目，就是构建这个未来的基石。🚀
