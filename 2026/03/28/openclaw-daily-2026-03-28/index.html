今日要点

🔥 OpenClaw 3月连发三个大版本（3.2 → 3.13 → 3.22 → 3.25），v2026.3.25 修复关键 bug；NVIDIA GTC 发布 NemoClaw 企业级方案；ClawHub 技能市场突破 13,700 个插件。

版本更新：v2026.3.25（3月25日发布）

继 v2026.3.22 这个”史诗级”更新后，OpenClaw 于 3 月 25 日推出了 v2026.3.25 修复版，主要解决了 3.22 引入的几个关键问题：

核心变更

Gateway / OpenAI 兼容性大幅增强

  • 新增 /v1/models 和 /v1/embeddings 端点
  • /v1/chat/completions 和 /v1/responses 支持显式模型覆盖转发
  • 对 RAG 应用和第三方客户端的兼容性显著提升
  • 贡献者：@vincentkoc

Microsoft Teams 全面升级

  • 迁移至官方 Teams SDK
  • 新增流式 1:1 回复、欢迎卡片、反馈/反思机制、状态更新、输入指示器、原生 AI 标签
  • 支持消息编辑和删除（含线程内回退）
  • 来源：PR #51808

Control UI 全面重构

  • 技能管理新增状态过滤标签（全部 / 就绪 / 需配置 / 已禁用）
  • Agent workspace 文件支持展开预览 Markdown
  • 配置页面改为可折叠树形侧边栏
  • 贡献者：@BunsDev

其他亮点

  • 🐳 --container 参数：直接在 Docker/Podman 容器内执行 OpenClaw 命令
  • 💬 Discord 自动线程支持 LLM 生成标题（autoThreadName: "generated"）
  • 🔌 插件钩子 before_dispatch 新增规范化入站元数据
  • 📦 内置技能添加一键安装配方（coding-agent、gh-issues、weather 等）

升级命令

⚠️ 注意：v2026.3.22 有 13 个 Breaking Changes，从旧版本升级前务必备份配置。运行 openclaw doctor --fix 可自动迁移浏览器配置等。

NVIDIA NemoClaw：企业级 AI Agent 基础设施

3 月 16 日 GTC 大会上，黄仁勋亲自宣布 NVIDIA NemoClaw——为 OpenClaw 量身定制的企业级运行时栈：

一行命令部署

核心能力

  • 🔒 OpenShell 沙箱：隔离执行环境，策略化安全/网络/隐私护栏
  • 🧠 Nemotron 本地模型：支持 RTX PC、DGX Spark 本地推理
  • ☁️ 隐私路由器：本地+云端模型混合调度，敏感数据不出本地
  • 🏢 企业部署：从个人 Mac 到 DGX Station 全覆盖

黄仁勋评价：”Mac 和 Windows 是个人电脑的操作系统，OpenClaw 是个人 AI 的操作系统。”

来源：NVIDIA Newsroom

ClawHub 技能市场：13,700+ 插件生态

v2026.3.22 正式将 ClawHub 设为默认插件市场，告别以前四处找 GitHub 仓库的混乱时代：

覆盖领域：开发者工作流、Jira/GitHub 集成、智能家居（Home Assistant）、股票追踪、邮件摘要、日历管理等。

安全提醒：早期审计发现约 10.8% 的第三方插件存在恶意代码风险。ClawHub 内置安全扫描 + Silverfort ClawNet 安全层，但仍建议只安装来源可信的技能。

实战技巧：Sub-Agent 成本优化

v2026.3.22 新增了 子代理可调思维级别和多模型支持，这对控制成本至关重要：

省钱策略：

  1. 简单查询任务 → gpt-4o-mini + thinking: off
  2. 代码生成任务 → claude-sonnet-4 + thinking: low
  3. 复杂推理任务 → claude-opus-4 + thinking: high

据社区用户反馈，合理配置后 **月度 API 成本降低 40-60%**。
