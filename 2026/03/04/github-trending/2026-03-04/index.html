前言

今天的 GitHub Trending 榜单，一个词概括：Agent 疯了。从 AI 编码助手到 Agent 记忆管理，从数字伴侣到多 Agent 并行调度，10 个项目里至少 7 个跟 AI Agent 直接相关。另外还有一个用 WiFi 信号”透视”人体姿态的硬核项目杀入前三，着实让人眼前一亮。

来看看今天开源圈在搞什么。

1. ruvnet/RuView — 用 WiFi 信号”看见”人

项目简介

RuView 通过分析 WiFi 信道状态信息（CSI），实现无摄像头的人体姿态估计、生命体征监测（呼吸+心率）和穿墙感知。

为什么火？

今日 +4419 星，这个数字已经说明一切。隐私优先 + 无需穿戴设备 + 穿墙检测，这套组合拳直击智能家居和灾难救援的痛点。用 Rust 实现了 54K fps 的信号处理速度，ESP32 硬件成本仅 $54，技术壁垒和落地成本的反差感极强。

技术亮点

  • Rust 高性能信号处理管线，支持 ESP32-S3 边缘部署
  • 多基站网格（4-6 节点）实现 360° 覆盖，亚英寸精度
  • 自学习对比学习模型，无需标注数据即可跨环境迁移
  • QUIC 加密 Mesh 通信，端到端安全

适用场景

智能家居存在检测、老人跌倒监测、灾难搜救、隐私敏感场景的人体感知。

⭐ Stars: 25,870 | 📈 Today: +4,419 | Language: Rust

2. moeru-ai/airi — 你的赛博伴侣

项目简介

Project AIRI 是一个开源的 AI 虚拟角色平台，致力于复刻 Neuro-sama 的能力——实时语音聊天、玩 Minecraft/Factorio、Live2D 形象，全平台可用。

为什么火？

22K+ 星，Discord 社区活跃。它满足了一个真实需求：Neuro-sama 直播关了就没了，但 AIRI 让你拥有自己的数字生命。Web 技术栈（WebGPU/WebAudio/WASM）降低了部署门槛，桌面版则支持 CUDA/Metal 原生加速。

技术亮点

  • 基于 Web 技术但不牺牲性能，桌面版走 native GPU
  • 模块化架构：语音识别、TTS、强化学习、计算机视觉各自独立
  • PWA 支持移动端，Discord 语音频道接入

适用场景

个人数字伴侣、AI VTuber 开发、游戏 AI 伙伴研究。

⭐ Stars: 22,438 | 📈 Today: +832 | Language: TypeScript

3. K-Dense-AI/claude-scientific-skills — 给 AI Agent 装上科研大脑

项目简介

170+ 个开箱即用的科学研究 Agent 技能，覆盖生信、化学信息学、临床研究、材料科学等，兼容 Cursor/Claude Code/Codex。

为什么火？

把 AI 编码助手变成 AI 科学家，这个定位太准了。170 个技能、250+ 数据库接入、60+ Python 包的优化封装，每个都带完整文档和示例。对科研人员来说，这就是开箱即用的超级工具箱。

技术亮点

  • 遵循开放 Agent Skills 标准，跨平台兼容
  • 深度覆盖：基因组学、药物发现、蛋白质工程、量子计算、金融数据
  • 每个技能包含 SKILL.md + 代码示例 + 最佳实践

适用场景

生物医学研究、药物发现、临床数据分析、科研论文写作辅助。

⭐ Stars: 12,174 | 📈 Today: +798 | Language: Python

4. superset-sh/superset — 多 Agent 并行调度终端

项目简介

Superset 是一个专为编码 Agent 设计的”终端 IDE”，可同时运行 10+ 个 Claude Code/Codex 等 Agent，每个任务独立 git worktree 隔离。

为什么火？

+632 星。解决了一个很现实的问题：你有多个编码任务，想让多个 Agent 并行干活但互不干扰。Superset 提供 worktree 隔离、状态监控、内置 diff viewer，本质上是 Agent 时代的任务管理器。

技术亮点

  • 每个任务独立分支 + 工作目录，零冲突
  • 支持所有 CLI Agent：Claude Code、Codex、Gemini CLI、Cursor Agent 等
  • 内置变更审查和编辑器，一站式工作流

适用场景

需要同时处理多个开发任务的团队/个人，Agent 密集型开发工作流。

⭐ Stars: 4,354 | 📈 Today: +632 | Language: TypeScript

5. msitarzewski/agency-agents — AI Agent 角色模板库

项目简介

一套精心设计的 AI Agent 人格模板集合，涵盖前端开发、后端架构、UX 设计、增长黑客、社区运营等 30+ 个专业角色。

为什么火？

起源于 Reddit 热帖，经过数月迭代。每个 Agent 不是简单的 prompt template，而是有完整人格、工作流程和可交付物定义的”虚拟员工”。直接复制到 Claude Code 就能用。

技术亮点

  • 四大部门：工程、设计、营销、产品，覆盖完整团队角色
  • 每个角色有独特的沟通风格、成功指标、代码模板
  • 即插即用，兼容 Claude Code

适用场景

团队 AI 赋能、个人效率提升、Claude Code 深度用户。

⭐ Stars: 3,840 | 📈 Today: +593 | Language: -

6. agentscope-ai/agentscope — 生产级 Agent 框架

项目简介

AgentScope 是一个面向生产环境的 Agent 框架，提供 ReAct Agent、工具调用、人机协同、记忆、规划、实时语音等核心能力。

为什么火？

阿里达摩院出品，设计理念是顺应模型能力增长而非用死板的 prompt 约束模型。内置 MCP/A2A 协议支持、OTel 可观测性、K8s 部署，真正面向生产。

技术亮点

  • 内置 ReAct Agent + MCP/A2A 协议 + 消息中枢
  • 支持 Agent 微调（Trinity-RFT 强化学习）
  • 2026年2月新增实时语音 Agent 支持

适用场景

企业级 Agent 应用开发、多 Agent 编排、需要可观测性和部署能力的团队。

⭐ Stars: 17,144 | 📈 Today: +112 | Language: Python

7. LMCache/LMCache — LLM 推理的 KV 缓存加速层

项目简介

LMCache 为 LLM 推理引擎提供 KV Cache 缓存层，通过在 GPU/CPU/磁盘/S3 间存储和复用 KV Cache，将首 token 延迟降低 3-10 倍。

为什么火？

被 vLLM、SGLang、NVIDIA Dynamo、Google Cloud、CoreWeave 等主流平台采用或集成。在长上下文场景下效果显著，RAG 和多轮对话场景直接受益。

技术亮点

  • 支持 vLLM v1 集成：CPU 卸载、分离式 prefill、P2P 共享
  • 跨实例复用任意文本（不限前缀）的 KV Cache
  • 支持 NIXL 高性能传输

适用场景

LLM 推理服务优化、长上下文应用、RAG 系统加速。

⭐ Stars: 7,457 | 📈 Today: +135 | Language: Python

8. agentscope-ai/ReMe — Agent 记忆管理套件

项目简介

ReMe 是专为 AI Agent 设计的记忆管理框架，提供文件式和向量式两种记忆系统，解决上下文窗口有限和会话无状态两大核心问题。

为什么火？

Agent 跑得再好，没记忆就是金鱼。ReMe 的文件式记忆（MEMORY.md + 每日日志）简单直观，向量式记忆支持个人/任务/工具三维度检索。跟 AgentScope 生态深度集成。

技术亮点

  • 文件式记忆：Markdown 文件即记忆，可读可编辑可迁移
  • 向量式记忆：混合检索（向量 + BM25）
  • 自动压缩长对话、后台异步摘要

适用场景

任何需要长期记忆的 Agent 应用、个人助手、任务型 Agent。

⭐ Stars: 1,388 | 📈 Today: +49 | Language: Python

9. CodebuffAI/codebuff — 多 Agent 协作编码助手

项目简介

Codebuff 是一个开源终端编码助手，通过协调多个专业 Agent（文件选择器、规划器、编辑器、审查器）来理解和修改代码库。

为什么火？

在 175+ 编码任务评测中以 61% vs 53% 击败 Claude Code。多 Agent 架构比单模型方法带来更好的上下文理解和更少的编辑错误。支持自定义 Agent 定义，可扩展性强。

技术亮点

  • 四阶段 Agent 流水线：定位 → 规划 → 编辑 → 审查
  • TypeScript Agent DSL，支持自定义 Agent 和工具
  • npm 一键安装，即装即用

适用场景

日常编码辅助、代码库重构、需要精确编辑的开发任务。

⭐ Stars: 3,353 | 📈 Today: +126 | Language: TypeScript

10. aquasecurity/trivy — 全能安全扫描器

项目简介

Trivy 是一个综合安全扫描器，可扫描容器镜像、文件系统、Git 仓库、K8s 集群，检测漏洞、配置错误、密钥泄露和许可证问题。

为什么火？

Aqua Security 出品的老牌安全工具今天又上榜，说明安全需求从未减退。一个工具覆盖 SBOM、CVE、IaC 错误配置、Secrets 扫描，开箱即用，CI/CD 集成简单。

技术亮点

  • 扫描目标全覆盖：容器、文件系统、Git、VM、K8s
  • 支持主流语言和操作系统
  • GitHub Actions、K8s Operator、VS Code 插件生态完善

适用场景

CI/CD 安全门禁、容器安全、合规检查、DevSecOps。

⭐ Stars: 617 | 📈 Today: +164 | Language: Go

总结

今天的 Trending 榜单有三个明显趋势：

  1. Agent 基础设施井喷：从框架（AgentScope）到记忆（ReMe）到调度（Superset）到编码（Codebuff），Agent 开发的每一层都有新工具涌现。这不是炒概念，是真正的工程化落地。

  2. WiFi 感知的破圈：RuView 用 WiFi 信号做人体姿态估计，+4400 星的爆发说明隐私优先的感知技术有巨大需求。当摄像头让人不安时，无处不在的 WiFi 成了新的感知介质。

  3. AI 科研工具民主化：claude-scientific-skills 把 170 个科研技能打包成开箱即用的 Agent 技能包，降低了 AI 辅助科研的门槛。

Agent 时代不是在来的路上——它已经到了。
