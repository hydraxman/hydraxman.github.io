🌍 国际要闻

1. 美军驻科威特基地遭伊朗导弹袭击，6名美军阵亡

来源：BBC World

美国国防部长证实，伊朗于周日对美军在科威特的军事基地发动导弹袭击，造成6名美军士兵死亡。中东冲突持续升级。

2. 美伊战争引发中东各国连锁反应

来源：AP News

美以联合对伊朗军事行动持续，伊朗封锁霍尔木兹海峡导致全球油气价格飙升，地区安全形势急剧恶化。

3. 鲁比奥关于以色列参与攻伊言论引发政治风暴

来源：Al Jazeera

国务卿鲁比奥暗示以色列在美国对伊军事行动中扮演角色，在美国各政治派别引发巨大反响，战争正当性受到质疑。

4. 特朗普威胁因军事基地问题与西班牙断绝贸易

来源：BBC World

西班牙禁止美国使用其军事基地后，特朗普发出贸易威胁，加剧美欧盟友关系紧张。

5. 马克龙警告以色列不要入侵黎巴嫩

来源：Al Jazeera

真主党对以色列发动袭击后，以色列对黎巴嫩展开新一轮轰炸。法国总统呼吁以色列保持克制。

🌐 科技新闻精选

1. OpenAI 正在开发 GitHub 替代平台

来源：Slashdot

据 The Information 报道，OpenAI 正在开发代码托管平台，可能与微软 GitHub 直接竞争。

2. 阿里巴巴通义千问技术负责人林骏扬离职

来源：TechCrunch

Qwen 团队技术负责人林骏扬在重大模型发布后离职，在中国AI圈引发震荡。

3. LLM 可以大规模揭示匿名用户身份

来源：Ars Technica

研究发现大语言模型能以惊人准确率大规模去匿名化，网络假名隐私保护可能形同虚设。

4. Google Chrome 将改为两周发布周期

来源：Slashdot

从9月 Chrome 153 开始，大版本发布频率从四周缩短至两周。

5. 150万用户离开 ChatGPT

来源：Reddit Technology

ChatGPT 出现显著用户流失，用户对 OpenAI 近期争议行为表达不满。

6. Apple 发布搭载 M5 芯片的新 MacBook 系列

来源：Wired / Gadgets360 / HN

包括 M5 MacBook Air、M5 Pro/Max MacBook Pro、iPhone 17e 等密集发布。

7. 美国政府 iPhone 入侵工具包落入犯罪分子手中

来源：Wired / TechCrunch

高级 iPhone exploit toolkit 现已被网络犯罪分子和外国间谍使用，可能已感染数万台设备。

8. Meta AI 眼镜将私密视频发送给人工审核员

来源：Slashdot / 9to5Mac

欧洲 Meta 智能眼镜用户的敏感视频可能被发送给人工数据标注员审核，引发隐私争议。
