GitHub Trending 日报：2026年3月5日最火的 10 个开源项目深度解析

今天的 GitHub Trending 堪称「AI Agent 大阅兵」。十个项目里有八个直接和 AI Agent 相关，剩下两个也在为 AI 基础设施添砖加瓦。让我们逐个深度剖析。

1. Shannon — AI 自动化渗透测试器

📦 KeygraphHQ/shannon | TypeScript | ⭐ 30,825 | 📈 +1,854 today

Shannon 是一款完全自主的白盒 AI 渗透测试工具。它能分析你的源代码、识别攻击向量，然后执行真实的漏洞利用来证明安全问题。在 XBOW 基准测试上达到了 96.15%（104 个漏洞中成功利用 100 个）的命中率。

为什么火？ AI 编程工具让团队每天都在发布代码，但渗透测试通常一年才做一次。Shannon 填补了这个巨大的安全缺口——每次构建都能跑一遍自动化渗透测试。它不报告”理论上可能存在的漏洞”，只输出有完整 PoC 的已验证漏洞。

技术亮点： 结合源代码分析和浏览器自动化进行动态测试，支持 SQL 注入、XSS、SSRF、认证绕过等 OWASP 全品类攻击，可并行执行多个漏洞类别的分析和利用。Pro 版还集成了 SAST、SCA 和密钥扫描。

适用场景： 有 CI/CD 流水线的团队，想在每次发布前自动跑安全测试。AGPL-3.0 开源，本地白盒测试免费用。

2. The Agency — AI Agent 人设模板库

📦 msitarzewski/agency-agents | ⭐ 5,765 | 📈 +2,209 today

一个精心编写的 AI Agent 人设集合，涵盖前端开发、后端架构、UI 设计、增长黑客、Reddit 社区运营等数十个专业角色。每个 Agent 都有独立的性格、工作流程和交付标准。

为什么火？ 今日涨星最多的项目。它解决了一个真实痛点：大家都知道 AI Agent 好用，但写好一个 Agent 的 system prompt 很费脑子。这个库提供了「即插即用」的专业 Agent 模板，直接复制到 Claude Code 就能用。

技术亮点： 不是简单的 prompt 模板，而是完整的角色定义——包含身份特征、核心使命、技术交付物（含代码示例）、成功指标和沟通风格。分工程、设计、营销、产品四大部门，像搭建一个虚拟团队。

适用场景： 独立开发者或小团队想快速组建 AI 协作团队，或者想学习如何写高质量 Agent prompt 的人。

3. Trivy — 云原生安全扫描瑞士军刀

📦 aquasecurity/trivy | Go | ⭐ 914 | 📈 +380 today

Trivy 是一款全方位安全扫描器，能扫描容器镜像、文件系统、Git 仓库、K8s 集群等目标，查找 CVE 漏洞、IaC 配置错误、密钥泄露和 SBOM。

为什么火？ 安全话题持续升温。Trivy 一条命令就能完成过去需要多个工具才能做的事。和上面的 Shannon 一起出现在 Trending 说明安全工具正迎来一波关注潮。

技术亮点： 支持几乎所有主流语言和平台，有 GitHub Actions、K8s Operator、VS Code 插件等生态集成。安装简单（brew install trivy），使用更简单（trivy image python:3.4-alpine）。

适用场景： 所有需要在 CI/CD 中集成安全扫描的团队。和 Shannon 的区别是：Trivy 侧重漏洞扫描和合规检查，Shannon 侧重渗透测试和漏洞利用验证。

4. Claude Scientific Skills — 170+ 科研 Agent 技能包

📦 K-Dense-AI/claude-scientific-skills | Python | ⭐ 12,896 | 📈 +940 today

一个包含 170+ 科研技能的开源集合，覆盖生物信息学、药物发现、蛋白质组学、临床研究、材料科学、天文学等领域。支持 250+ 科学数据库访问。

为什么火？ 把 AI 编程助手变成「AI 科学家」。科研工作者不需要自己查 API 文档、写集成代码，直接用自然语言就能执行复杂的多步科学工作流。支持 Cursor、Claude Code、Codex 等主流 AI 编码工具。

技术亮点： 每个技能都包含完整文档、实用代码示例、最佳实践和集成指南。从基因组分析到分子对接，从临床试验到量子计算，覆盖面惊人。遵循开放的 Agent Skills 标准。

适用场景： 生物医药、化学、物理等领域的研究人员。特别适合需要跨学科数据分析的场景。

5. Codebuff — 多 Agent 协作的 AI 编程助手

📦 CodebuffAI/codebuff | TypeScript | ⭐ 3,625 | 📈 +337 today

一个开源 AI 编码助手，用多 Agent 架构替代单模型方案。在 175+ 编码任务的评测中以 61% vs 53% 击败 Claude Code。

为什么火？ 多 Agent 编码是当前热门方向。Codebuff 把一个编码请求拆分给文件选择、规划、编辑、审查四个专门的 Agent 协作完成，每个 Agent 做自己最擅长的事。还支持自定义 Agent 定义，可扩展性很强。

技术亮点： Agent 定义文件支持 TypeScript，可以混合声明式和编程式控制。比如示例中的 git-committer Agent，先通过工具调用分析 git 状态，再由 LLM 决定提交信息。架构优雅。

适用场景： 对 Claude Code / Cursor 效果不满意的开发者，或想构建自定义编码工作流的团队。

6. OpenSandbox — 阿里巴巴开源的 AI 沙箱平台

📦 alibaba/OpenSandbox | Python | ⭐ 6,120 | 📈 +788 today

一个通用 AI 沙箱平台，提供多语言 SDK、统一沙箱 API 和 Docker/K8s 运行时，支持编码 Agent、GUI Agent、Agent 评估、AI 代码执行和强化学习训练等场景。

为什么火？ AI Agent 越来越需要安全的执行环境。OpenSandbox 提供了一个标准化的沙箱方案，让 Agent 可以安全地执行代码、操作浏览器、运行桌面环境，而不会影响宿主系统。阿里巴巴出品，生产级品质。

技术亮点： 支持 Python、Java、TypeScript、C#、Go 等多语言 SDK。内置网络策略（入口网关 + 出口控制），支持 Chrome/Playwright 浏览器自动化和 VNC 桌面环境。K8s 运行时支持大规模分布式调度。

适用场景： 构建需要代码执行能力的 AI Agent，或需要安全沙箱环境进行 Agent 评估的团队。

7. MoneyPrinterV2 — AI 自动化赚钱工具

📦 FujiwaraChoki/MoneyPrinterV2 | Python | ⭐ 14,333 | 📈 +143 today

自动化在线赚钱流程的工具集，包括 Twitter 机器人、YouTube Shorts 自动生成、亚马逊联盟营销和本地商家外展。

为什么火？ 「AI 赚钱」永远是流量密码。这是 V1 的完全重写版，更模块化的架构支持定时任务调度。虽然名字有点噱头，但核心是 AI 内容自动化生产和分发。

技术亮点： 支持 CRON 调度的 Twitter 和 YouTube 自动化，亚马逊联盟营销集成，本地商家爬取和邮件外展。需要 Python 3.12。

适用场景： 想用 AI 自动化内容创作和分发的副业玩家。注意：自动化内容创作有平台合规风险，请谨慎使用。

8. Perplexica — 隐私优先的 AI 搜索引擎

📦 ItzCrazyKns/Perplexica | TypeScript | ⭐ 30,990 | 📈 +1,090 today

一个可以完全在你自己硬件上运行的隐私优先 AI 搜索引擎，支持本地 LLM（Ollama）和云端模型（OpenAI、Claude、Groq），提供带引用来源的精准回答。

为什么火？ Perplexity 的开源替代品，解决了 AI 搜索的隐私问题。3 万 star 说明市场对隐私搜索的需求很大。一条 Docker 命令就能跑起来，内置 SearxNG 搜索引擎。

技术亮点： 三种搜索模式（速度/平衡/深度研究），支持文件上传问答，智能搜索建议，搜索历史本地保存。支持限定域名搜索，特别适合技术文档查阅。

适用场景： 关注数据隐私的个人或企业，想搭建自己的 AI 搜索引擎。

9. AgentScope — 生产就绪的 Agent 框架

📦 agentscope-ai/agentscope | Python | ⭐ 17,475 | 📈 +427 today

一个生产就绪、易用的 Agent 框架，内置 ReAct Agent、工具调用、人机协作、记忆、规划、实时语音、评估和模型微调支持。

为什么火？ Agent 框架竞争激烈，AgentScope 的差异化在于「面向日益增强的模型能力设计」——不用严格的 prompt 和固定的编排约束模型，而是充分利用模型自身的推理和工具使用能力。支持 MCP 和 A2A 协议，生态集成丰富。

技术亮点： 2026 年 2 月刚加入实时语音 Agent 支持，支持数据库记忆和记忆压缩，支持 Docker/K8s 部署和 VNC GUI 沙箱。还有配套的强化学习微调库 Trinity-RFT。

适用场景： 需要构建生产级多 Agent 系统的团队，特别是需要语音交互、长期记忆和模型微调的场景。

10. Project AIRI — 开源 AI 虚拟伴侣

📦 moeru-ai/airi | TypeScript | ⭐ 25,515 | 📈 +1,454 today

受 Neuro-sama 启发的开源 AI 虚拟伴侣项目，支持实时语音聊天、玩 Minecraft 和 Factorio，Web/macOS/Windows 全平台运行。

为什么火？ 2.5 万 star，今日 +1,454，说明「AI 虚拟伴侣」赛道热度不减。与 Character.ai 等平台的区别是：AIRI 不只能聊天，还能看你写代码、和你一起玩游戏、做直播。自托管，数据完全属于你。

技术亮点： 大量使用 Web 技术（WebGPU、WebAudio、WebAssembly）实现浏览器端运行，桌面版支持 CUDA 和 Metal 原生加速。Live2D/VRM 模型支持，PWA 移动端适配。有专门的子项目组织 @proj-airi 负责 RAG、记忆系统等模块。

适用场景： 想拥有一个能陪你聊天、玩游戏、看你写代码的 AI 伴侣的开发者。也适合想研究 AI 虚拟角色技术栈的人。

总结

今天的 Trending 呈现三个明显趋势：

  1. AI 安全工具爆发：Shannon 和 Trivy 同时上榜，AI 在加速开发的同时也在加速安全防护。
  2. Agent 框架百花齐放：AgentScope、Codebuff、OpenSandbox 分别在框架、编码和基础设施层面推进 Agent 生态。
  3. AI 从工具走向伙伴：Project AIRI 代表了一个方向——AI 不只是完成任务的工具，而是能陪伴你的存在。

AI Agent 的 2026，才刚刚开始。
