前言

今天的 GitHub Trending 榜单，AI Agent 框架几乎霸占了一半席位。从字节跳动的 DeerFlow 2.0 到金融研究 Agent Dexter，再到 Claude Code 多智能体编排工具 oh-my-claudecode，Agent 生态正以前所未有的速度膨胀。更有 RuView 用 WiFi 信号实现人体姿态估计，堪称”物理黑科技”。一起来看今日最火的 9 个开源项目。

1. bytedance/deer-flow — 字节出品的超级 Agent 框架

⭐ 48,731 | 📈 +2,394 today | Language: Python

DeerFlow 2.0 是字节跳动开源的长时间任务超级 Agent 框架，从零重写，与 1.x 完全不同。它编排子智能体、记忆、沙箱和技能插件，能处理从几分钟到数小时的复杂任务——研究、编码、内容创作一条龙。

为什么火？

2.0 版本是 ground-up rewrite，架构上脱胎换骨。字节背书 + 与火山引擎 Doubao-Seed-2.0-Code 深度整合，让它不只是玩具，而是有生产级野心的项目。2月底首次登上 GitHub Trending 第一名后热度不减。

技术亮点

  • 子智能体编排：任务分解、委派、聚合的完整 pipeline
  • 沙箱执行：代码在隔离环境中运行，安全可控
  • 记忆系统：跨任务上下文保持，长期记忆持久化
  • 技能扩展：插件化技能系统，社区可贡献
  • 推荐搭配 Doubao-Seed-2.0-Code、DeepSeek V3.2、Kimi 2.5 使用

适用场景

需要长时间自主运行的复杂研究、代码生成、内容创作任务。适合企业级 AI 工作流搭建。

2. mvanhorn/last30days-skill — AI Agent 的全网研究技能

⭐ 10,520 | 📈 +2,685 today | Language: Python

这是一个 Claude Code / AI Agent 的”研究技能插件”：给它一个话题，它会自动搜索 Reddit、X、YouTube、Hacker News、Polymarket 和 Web，然后合成一份有理有据的摘要报告。

为什么火？

解决了一个真实痛点：AI Agent 怎么快速了解一个话题的最新舆论？ 它不是简单搜索，而是跨 8 个信号源聚合，还加入了 Polymarket 预测市场数据（人们用真金白银下注的预测），比纯文本搜索可靠得多。

技术亮点

  • 集成 Reddit（含热门评论加权）、Instagram Reels、TikTok 等 8 个数据源
  • ScrapeCreators 统一 API：一个 key 覆盖 Reddit + TikTok + Instagram
  • 自动保存研究结果到 ~/Documents/Last30Days/，建立个人研究库
  • 455+ 测试用例，工程质量扎实

适用场景

竞品调研、舆情监控、投资决策前的快速信息扫描。Claude Code 用户装上就能用。

3. ruvnet/RuView — WiFi 信号透视人体姿态

⭐ 43,277 | 📈 +1,002 today | Language: Rust

这个项目堪称”物理学的胜利”：用 WiFi 信号实现人体姿态估计、呼吸/心率监测和存在检测，完全不需要摄像头。基于 CMU 的 DensePose From WiFi 论文，但做成了可部署的边缘计算系统。

为什么火？

“用 WiFi 穿墙看人”这个概念本身就足够震撼。更关键的是它用 ~$1 的 ESP32 节点就能运行，不需要云端、不需要摄像头、不需要互联网。隐私友好的感知方案，想象空间巨大。

技术亮点

  • 分析 WiFi CSI（信道状态信息）扰动来重建人体位置
  • 基于 RuVector 自学习向量记忆系统
  • 纯 Rust 实现，边缘设备部署
  • 1300+ 测试用例，Docker 多架构支持
  • 系统自主学习房间 RF 特征，无需标注数据

适用场景

智能家居存在检测、老人跌倒监测、安防系统。隐私敏感场景的最佳选择——没有摄像头，只有物理信号。

4. Vaibhavs10/insanely-fast-whisper — 极速语音转写

⭐ 11,324 | 📈 +1,370 today | Language: Python

顾名思义：疯狂快的 Whisper。150 分钟音频在 A100 上仅需 98 秒完成转写。基于 HuggingFace Transformers + Flash Attention 2 优化。

为什么火？

Whisper 本身很好用但很慢，这个项目把速度优化到了极致。一行 pipx install 加一行命令就能用，开箱即用的体验让它在社区持续获得关注。

技术亮点

  • Flash Attention 2 加持，fp16 + batch=24 极限优化
  • 支持 distil-large-v2，速度更快精度略有折中
  • 简洁 CLI，支持本地文件和 URL 输入
  • macOS MPS 支持

适用场景

播客转录、会议纪要、视频字幕生成。需要高吞吐批量处理音频的场景。

5. Yeachan-Heo/oh-my-claudecode — Claude Code 多智能体编排

⭐ 12,791 | 📈 +598 today | Language: TypeScript

Claude Code 的”oh-my-zsh”——一套零学习成本的多智能体编排工具。输入 autopilot: build a REST API，它自动规划、执行、验证、修复。

为什么火？

Claude Code 火了之后，围绕它的工具链生态开始爆发。OMC 的 Team Mode（v4.1.7+）提供了结构化的多 Agent 协作：plan → PRD → exec → verify → fix 循环。对于大型项目开发，这比单 Agent 可靠得多。

技术亮点

  • Team Mode：多 Agent 分阶段 pipeline，支持 Claude Code 原生 teams
  • Deep Interview：苏格拉底式追问，在写代码前澄清需求
  • 支持 Claude Code Plugin Marketplace 一键安装
  • 多语言文档（中日韩英西法葡越）

适用场景

使用 Claude Code 进行团队协作开发。需求不清晰时用 deep-interview 先梳理，然后 team mode 执行。

6. datalab-to/chandra — 复杂文档 OCR 之王

⭐ 6,237 | 📈 +557 today | Language: Python

Chandra OCR 2 是一个处理复杂表格、表单、手写体的 OCR 模型，输出结构化 HTML/Markdown/JSON，保留完整版面信息。支持 90+ 语言。

为什么火？

在 olmocr 基准测试中拿到第一。多语言性能是其重点投入方向，表格、数学公式、复杂排版的处理效果远超一般 OCR 工具。

技术亮点

  • 支持 vLLM 推理（推荐）和 HuggingFace 本地推理两种模式
  • 手写体、复选框表单精准识别
  • 图表提取并自动生成描述
  • 一行 pip install chandra-ocr 即可使用

适用场景

扫描件数字化、合同/发票解析、多语言文档处理、学术论文 OCR。

7. agentscope-ai/agentscope — 可观测的 Agent 框架

⭐ 20,547 | 📈 +437 today | Language: Python

AgentScope 的定位是”Build agents you can see, understand and trust“。支持 MCP、A2A 协议，内置 ReAct Agent、工具调用、记忆、规划、实时语音、评估和模型微调。

为什么火？

在 Agent 框架百花齐放的当下，AgentScope 强调的是生产可用和可观测性。内置 OTel 支持、K8s 部署、Serverless 方案，不是实验品而是工程工具。

技术亮点

  • 内置 MCP 和 A2A 协议支持
  • Message Hub 实现灵活的多 Agent 编排
  • 5 分钟快速上手，提供 ReAct Agent 开箱即用
  • 支持 Agent 微调（fine-tuning），不只是 prompt engineering

适用场景

企业级 Agent 应用部署。需要监控、审计、扩展能力的正式生产环境。

8. virattt/dexter — 金融研究自主 Agent

⭐ 19,052 | 📈 +210 today | Language: TypeScript

Dexter 是一个自主金融研究 Agent：给它一个复杂的金融问题，它自动分解成研究步骤，调用实时市场数据 API，自我验证并迭代直到得出可靠结论。

为什么火？

“Claude Code for Finance”的定位精准击中了金融从业者的痛点。内置循环检测和步数限制防止失控，自我验证机制提升了可信度。最近还加了 WhatsApp 集成。

技术亮点

  • 任务自动分解 + 工具选择 + 自我验证循环
  • 接入 Financial Datasets API（财报、资产负债表、现金流量表）
  • 内置安全机制：loop detection + step limits
  • 基于 Bun 运行时，TypeScript 全栈

适用场景

投资研究、财务分析、企业尽职调查。适合需要深度金融数据分析的个人投资者和分析师。

9. twentyhq/twenty — 开源 CRM 挑战 Salesforce

⭐ 41,334 | 📈 +117 today | Language: TypeScript

Twenty 是社区驱动的开源 CRM，目标是成为 Salesforce 的现代替代品。借鉴了 Notion、Airtable、Linear 的 UX 设计理念。

为什么火？

CRM 是万亿市场，而 Salesforce 太贵且锁定用户数据。Twenty 用开源打破垄断，UI/UX 设计现代化，支持自定义对象、字段、权限、工作流自动化。持续稳定在 Trending 上说明社区认可度很高。

技术亮点

  • 自定义对象和字段，灵活适配业务
  • 基于角色的权限管理
  • 工作流自动化（触发器 + 动作）
  • 邮件、日历、文件集成
  • Docker Compose 一键自部署

适用场景

中小企业 CRM 需求，不想被 Salesforce 绑架的团队。有技术能力自部署的组织。

总结

今日 Trending 的关键词是 Agent 和 边缘智能：

  1. Agent 框架内卷加剧：DeerFlow 2.0、AgentScope、oh-my-claudecode 三个 Agent 框架同时上榜，赛道极度拥挤
  2. 垂直 Agent 崛起：Dexter（金融）、last30days-skill（研究）证明通用框架之外，垂直场景更容易出彩
  3. 硬核物理感知：RuView 用 WiFi 做人体感知，提醒我们 AI 不只是 LLM
  4. 基础设施持续进化：insanely-fast-whisper 和 Chandra OCR 2 在语音和视觉领域持续提升速度和精度

明天见！🦌
