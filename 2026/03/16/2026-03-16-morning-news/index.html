🌍 国际要闻

1. 美以联军空袭伊朗哈马丹市

美以对伊朗哈马丹发动大规模空袭，现场浓烟滚滚。伊朗外长阿拉格奇表示德黑兰从未寻求停火，已准备长期作战。特朗普声称双方正在谈判，遭伊方否认。

2. 美军加油机在伊拉克坠毁，6名军人殉职

一架美军KC-135加油机在伊拉克执行任务时坠毁，机上6名军人全部遇难。

3. 伊朗战争耗资已达120亿美元

特朗普高级顾问透露对伊作战已花费120亿美元，战争终局仍不明朗，国内压力持续升高。与此同时，特朗普呼吁盟国护航霍尔木兹海峡船只，响应者寥寥。

4. 肯尼亚严重洪灾致66人死亡

内罗毕及周边地区持续暴雨引发严重洪灾，已造成至少66人死亡，救援行动仍在进行。

5. 泽连斯基指责欧盟”勒索”乌克兰

乌克兰总统泽连斯基指责部分欧盟国家以石油管道问题施压，称恢复俄罗斯经乌向欧盟输送石油等同于解除对俄制裁。

🌐 科技新闻

1. 字节跳动暂停 Seedance 2.0 全球发布

字节跳动因法律合规问题推迟其视频生成AI Seedance 2.0的全球上线，工程师和律师团队正在紧急处理潜在法律风险。

2. Meta 据报裁员高达20%

据路透社报道，Meta正计划大规模裁员（最多20%），以抵消在AI和数据中心方面的巨额支出。

3. Aether OS：浏览器中的 AT Protocol 操作系统

Aether OS 在浏览器中构建完整桌面环境，直接接入Bluesky底层的AT Protocol，支持去中心化应用，是Web3和社交协议融合的有趣尝试。

4. GitHub 供应链攻击：隐形 Unicode 代码注入

攻击者利用人眼不可见的Unicode字符在GitHub等代码仓库中植入恶意代码，这种”隐形代码”曾被放弃使用，如今被恶意利用。

5. 英国投资25亿英镑建设核聚变产业

英国科学部长宣布五年25亿英镑的核聚变投资计划，目标是在全球清洁能源竞赛中抢占先机。

6. AI 心理伤害诉讼律师警告大规模伤亡风险

代理AI致人自杀案件的律师表示，AI聊天机器人已开始出现在大规模伤亡事件中，呼吁加强监管。

7. Google 320亿美元收购 Wiz 内幕

Index Ventures的Shardul Shah详解了Google史上最大收购案——320亿美元收购云安全公司Wiz的幕后故事。

8. Chrome DevTools MCP 登上 Hacker News

Chrome DevTools 推出 MCP（Model Context Protocol）支持，让AI工具可以直接操控浏览器开发者工具。
