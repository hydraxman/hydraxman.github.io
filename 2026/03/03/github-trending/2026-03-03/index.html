前言

2026 年 3 月 3 日，GitHub Trending 热榜再次给我们带来了不少惊喜。今天的趋势可以用三个关键词概括：感知革命、AI Agent 编排、开发者工具升级。从用 WiFi 信号”看穿墙壁”的 RuView，到阿里开源的通用沙箱平台 OpenSandbox，每个项目都在推动技术边界。让我们逐一拆解。

1. RuView — WiFi 信号也能做人体姿态估计

仓库： ruvnet/RuView

项目简介

RuView 利用普通 WiFi 信号的信道状态信息（CSI），实现实时人体姿态估计、呼吸心率检测和存在感知——完全不需要摄像头。

为什么火？

隐私友好的感知方案是刚需。传统方案离不开摄像头，而 RuView 用无线电波替代了像素，让”无摄像头监控”成为现实。今日 +5096 星，说明社区对这个方向的渴望。

技术亮点

  • Rust 实现，处理速度达 54K fps
  • 支持 ESP32-S3 硬件流水线，成本低至 ~$54
  • 呼吸检测精度 6-30 BPM，心率 40-120 BPM
  • Docker 一键部署，30 秒上手

适用场景

智能家居、养老看护、安防监控等对隐私敏感的场景。需要注意的是，全功能需要 CSI 兼容硬件，普通 WiFi 只能做基础存在检测。

⭐ 22,978 Stars | 📈 +5,096 today | Language: Rust

2. Project AIRI — 开源版 Neuro-sama，你的 AI 虚拟伴侣

仓库： moeru-ai/airi

项目简介

一个自托管的 AI 虚拟角色容器，支持实时语音聊天、Minecraft/Factorio 游戏交互，覆盖 Web/macOS/Windows 全平台。

为什么火？

Neuro-sama 的爆火证明了 AI 虚拟角色的巨大市场需求，而 AIRI 把这件事开源了。自托管意味着完全掌控数据和角色人格，这对 vtuber 和二次元社区极具吸引力。

技术亮点

  • TypeScript 全栈，模块化灵魂容器架构
  • 实时语音交互 + 游戏操控双引擎
  • 多语言社区支持（中日韩英法俄越南）
  • Grok 模型集成，可自定义角色性格

适用场景

虚拟主播开发者、AI 伴侣研究、游戏 AI NPC 探索。

⭐ 21,618 Stars | 📈 +1,412 today | Language: TypeScript

3. Anthropic Prompt Engineering 交互式教程

仓库： anthropics/prompt-eng-interactive-tutorial

项目简介

Anthropic 官方出品的 9 章交互式提示工程教程，从基础结构到复杂工业级 prompt，手把手教你写好 Claude prompt。

为什么火？

官方教程永远是最权威的学习资源。这套教程从入门到高级层层递进，配有交互练习和答案，是目前最系统的 Claude prompt 工程学习路径。

技术亮点

  • 9 章 + 附录，覆盖角色分配、思维链、幻觉规避等核心技术
  • Jupyter Notebook 格式，边学边练
  • 附录涵盖链式 prompt、工具使用、检索增强等进阶主题

适用场景

所有使用 Claude API 的开发者，尤其是想从”能用”进阶到”好用”的 prompt 工程师。

⭐ 31,793 Stars | 📈 +526 today | Language: Jupyter Notebook

4. Ruflo — Claude 的企业级多智能体编排平台

仓库： ruvnet/ruflo

项目简介

前身是 Claude Flow，Ruflo 是一个为 Claude Code 打造的生产级多智能体协作框架，支持 60+ 专业 Agent 协同工作。

为什么火？

单 Agent 能力有上限，多 Agent 协作才是生产力的未来。Ruflo 把 Claude Code 变成了一个多智能体开发平台，支持分布式群体智能、容错共识和 RAG 集成。

技术亮点

  • WASM + Rust 内核驱动策略引擎
  • 自学习/自运维能力
  • 原生 Claude Code / Codex 集成
  • 5,800+ commits，成熟度可观

适用场景

需要多 Agent 协作的复杂软件工程任务，如大型代码库重构、跨模块开发。

⭐ 18,234 Stars | 📈 +830 today | Language: TypeScript

5. OpenSandbox — 阿里开源的 AI 通用沙箱平台

仓库： alibaba/OpenSandbox

项目简介

面向 AI 应用的通用沙箱平台，提供多语言 SDK、统一 API 和 Docker/K8s 运行时，覆盖 Coding Agent、GUI Agent、评测、代码执行等场景。

为什么火？

AI Agent 需要安全的代码执行环境，而现有方案要么太重要么不够通用。OpenSandbox 用统一抽象解决了这个问题，而且来自阿里的生产实践背书。今日 +1,026 星，增长迅猛。

技术亮点

  • 多语言 SDK：Python/Java/TypeScript/C#/Go
  • 沙箱协议标准化，支持自定义运行时扩展
  • 内置浏览器自动化（Chrome/Playwright）和桌面环境（VNC）
  • 网络策略：Ingress Gateway + 逐沙箱出口控制

适用场景

构建需要安全代码执行的 AI Agent、LLM 评测平台、强化学习训练环境。

⭐ 4,602 Stars | 📈 +1,026 today | Language: Python

6. MarkItDown — 微软的万能格式转 Markdown 工具

仓库： microsoft/markitdown

项目简介

微软出品的 Python 工具，将各种文件和办公文档转换为 Markdown 格式。

为什么火？

近 9 万星的老牌项目持续上榜，说明”把任何东西变成 Markdown”是一个永恒需求。在 RAG 和 LLM 数据预处理场景中，MarkItDown 几乎是标配。

技术亮点

  • 支持 Office 文档、PDF、HTML 等主流格式
  • 纯 Python 实现，易于集成
  • 持续维护，微软官方背书

适用场景

LLM 数据预处理、知识库建设、文档迁移。

⭐ 89,759 Stars | 📈 +648 today | Language: Python

7. Claude Scientific Skills — 148 个科学研究技能包

仓库： K-Dense-AI/claude-scientific-skills

项目简介

为 Claude Code/Cursor/Codex 等 AI Agent 提供 148+ 个即用型科学研究技能，覆盖生物、化学、医学、金融等领域。

为什么火？

AI Agent 的”技能市场”正在兴起。这个项目把专业领域的工作流打包成标准化技能，让 AI Agent 瞬间变身科研助手。250+ 数据库集成的覆盖面令人印象深刻。

技术亮点

  • 遵循开放 Agent Skills 标准
  • 覆盖基因组学、药物发现、临床研究、材料科学等
  • 包含金融/SEC 研究、对冲基金监控等非典型领域
  • 兼容 Cursor、Claude Code、Codex 多平台

适用场景

学术研究者、制药行业、金融分析师——任何需要 AI 辅助专业工作流的场景。

⭐ 11,270 Stars | 📈 +820 today | Language: Python

8. Superset — 编码 Agent 的超级终端

仓库： superset-sh/superset

项目简介

一个为 AI 编码 Agent 时代设计的终端 IDE，支持同时运行多个 Claude Code、Codex 等 Agent，每个任务隔离在独立的 git worktree 中。

为什么火？

当你同时跑 10 个 Agent 时，传统终端就不够用了。Superset 解决了多 Agent 并行开发的监控、隔离和切换问题——这是一个正在快速增长的真实需求。

技术亮点

  • 并行执行 10+ Agent，worktree 隔离互不干扰
  • 内置 diff 查看器，快速审查 Agent 改动
  • 工作区预设，自动化环境配置
  • 兼容所有 CLI Agent（Claude Code/Codex/Gemini CLI 等）

适用场景

重度使用 AI 编码 Agent 的开发者和团队，尤其是需要多任务并行的场景。

⭐ 3,644 Stars | 📈 +585 today | Language: TypeScript

9. Servo — Rust 编写的并行浏览器引擎

仓库： servo/servo

项目简介

Mozilla 孵化的原型浏览器引擎，用 Rust 编写，目标是为应用提供轻量、高性能的 Web 技术嵌入方案。

为什么火？

作为浏览器引擎赛道的”第三极”，Servo 代表了用系统语言重新思考 Web 渲染的可能性。虽然今日增长不算爆发（+45），但 35K+ 的总星数和持续活跃的开发说明社区对它的长期看好。

技术亮点

  • Rust 原生，内存安全 + 并行渲染
  • 支持 macOS/Linux/Windows/Android/OpenHarmony
  • 活跃的社区协作（Zulip + GitHub Issues）
  • 可嵌入应用的轻量级设计

适用场景

需要嵌入 Web 渲染的桌面/移动应用开发者，对浏览器引擎技术感兴趣的研究者。

⭐ 35,808 Stars | 📈 +45 today | Language: Rust

总结

今天的 Trending 榜单展现了三个清晰的趋势：

  1. 感知技术革新：RuView 用 WiFi 取代摄像头，隐私友好的感知方案正在成为现实。
  2. AI Agent 基础设施爆发：从 Ruflo 的多 Agent 编排到 OpenSandbox 的安全沙箱，再到 Superset 的 Agent 终端，围绕 AI Agent 的工具链正在快速成型。
  3. 知识民主化：Anthropic 的教程和 Claude Scientific Skills 降低了 AI 使用门槛，让更多人能高效利用 AI。

值得关注的是，Rust 语言在今天的榜单中表现抢眼（RuView + Servo），在性能敏感领域的采用率持续攀升。
