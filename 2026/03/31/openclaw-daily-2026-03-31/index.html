今日要点

OpenClaw 在 3 月末连续发版，v2026.3.29 带来 xAI/Grok 原生搜索整合、插件级工具审批钩子、ACP 会话频道绑定三大亮点。同时 ClawHub 技能注册表突破 13,000+ 技能，社区生态持续爆发。

版本更新：v2026.3.29（3月29日）

⚠️ Breaking Changes

Qwen Portal OAuth 移除 — 废弃了 qwen-portal-auth OAuth 集成。如果你还在用 portal.qwen.ai 的旧认证方式，需要迁移到 Model Studio：

旧配置迁移截止 — openclaw doctor 不再自动迁移两个月以前的旧配置 key，直接报验证失败。建议立即运行 openclaw doctor --fix 清理遗留配置。

🔥 重要新功能

1. xAI/Grok 原生搜索整合

OpenClaw 将 xAI 提供商迁移到 Responses API，新增 x_search 一等公民搜索工具。配置过 xAI key 的用户，搜索功能自动启用，无需手动开插件。还可以在 openclaw onboard 和 openclaw configure --section web 中选择 x_search 模型。

💡 实用建议：如果你同时配置了 Brave Search 和 xAI，现在可以按场景选择搜索后端——Brave 适合通用搜索，x_search 更适合实时社交媒体和新闻热点。

2. 插件工具审批钩子（requireApproval）

这是安全领域的重大进步。插件现在可以通过 before_tool_call 钩子暂停工具执行，弹出审批提示。支持：

  • exec 审批覆盖层
  • Telegram 按钮确认
  • Discord 交互式确认
  • 任意频道的 /approve 命令

/approve 命令现在统一处理 exec 和插件两种审批场景，自动回退。

💡 场景举例：你可以写一个插件，在 Agent 尝试发送邮件或删除文件时弹出确认，人工批准后才执行。这让 OpenClaw 在企业场景中的可控性大幅提升。

3. ACP 频道绑定（current-conversation）

Discord、BlueBubbles、iMessage 现在支持 ACP --bind here 模式，可以把当前聊天直接变成 Codex 工作空间，不再需要创建子线程。文档也明确区分了「聊天界面」「ACP 会话」和「运行时工作空间」三个概念。

4. MiniMax 图像生成

新增 MiniMax image-01 模型支持，包括文生图和图生图编辑，支持宽高比控制。对国内用户来说，又多了一个可用的图像生成后端。

5. 其他亮点

功能 说明
OpenAI apply_patch 默认启用 OpenAI/Codex 模型的 apply_patch 工具默认开启
Gemini CLI 后端 新增 Gemini CLI 推理后端，与 Claude CLI、Codex CLI 并列
Slack 文件上传 新增 upload-file 动作，支持文件名/标题/评论覆盖
Teams/Google Chat 文件统一 文件发送统一到 upload-file 动作
Matrix 语音消息 TTS 回复现在以原生 Matrix 语音气泡发送
openclaw config schema 新命令，打印 openclaw.json 的 JSON Schema
Podman 简化 容器部署简化为 rootless 用户模式

回顾：3月22日 v3.22 大版本

如果你还没升级到 3.22，这里是关键变更回顾：

  • ClawHub 原生集成 — openclaw skills install/search 一键安装社区技能，注册表已超 13,000 个技能
  • Matrix 插件 — 基于 matrix-js-sdk 的新消息插件，适合隐私优先和自托管场景
  • Anthropic Vertex AI — 通过 Google Cloud 运行 Claude，企业合规利器
  • 内置搜索提供商 — Exa、Tavily、Firecrawl 开箱即用
  • 移除 ClawdBot/MoltBot 遗留变量 — Breaking change，升级前必须更新环境变量

升级指南

升级注意事项：

  1. v2026.3.29 移除了 Qwen Portal OAuth，使用该认证方式的用户升级前先准备 Model Studio API key
  2. 两个月以上的旧配置不再自动迁移，建议先备份 openclaw.json
  3. 如果从 3.22 之前版本升级，需要先处理 ClawdBot/MoltBot 环境变量的重命名

社区动态

  • awesome-openclaw-skills 仓库（VoltAgent 维护）收录了 5,400+ 精选技能，按类别分类，是发现新技能的好入口
  • ClawHub 技能注册表新增原生包管理目录（code plugins + bundle plugins），技能可以声明依赖和版本要求
  • 社区讨论热点集中在 ACP 频道绑定的实际应用场景，以及 xAI 搜索与 Brave Search 的对比体验
