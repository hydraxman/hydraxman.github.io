前言

今天的 GitHub Trending 呈现出一个压倒性的主题：AI Agent 生态全面开花。从群体智能预测引擎到个人助手平台，从前端设计语言到 Agent 技能市场，几乎每个上榜项目都在回答同一个问题——如何让 AI 更有能力地帮人做事。值得注意的是，中国开发者的项目在今日榜单中表现抢眼。

1. GoogleCloudPlatform/generative-ai

项目简介

Google Cloud 官方的生成式 AI 示例代码库，覆盖 Gemini 3.1 Pro、Vertex AI Search、RAG、Imagen 等全套能力。

为什么火？

Gemini 3.1 Pro 刚刚发布，这个仓库是上手体验的第一站。Google 把所有 Notebook 和 Demo 集中在一处，开发者拿来即用。每次 Gemini 大版本更新，这个仓库都会冲上 Trending。

技术亮点

  • Jupyter Notebook 为主，覆盖文本、图像、视频、搜索、RAG 全模态
  • 与 Vertex AI 深度集成，适合企业级部署
  • 示例覆盖 function calling、grounding、多模态推理等前沿用法

适用场景

需要在 Google Cloud 上落地 GenAI 应用的企业开发者；想快速体验 Gemini 3.1 Pro 新特性的技术爱好者。

⭐ Stars: 15,340 | 📈 Today: +1,282 | Language: Jupyter Notebook

2. openclaw/openclaw

项目简介

跨平台个人 AI 助手，一个进程连接 20+ 消息平台（WhatsApp、Telegram、Slack、Discord、微信等），支持语音、Canvas、Cron 调度、技能系统。

为什么火？

29 万 star，今天单日 +9,164——这是一个持续爆发的怪物级项目。它解决了一个真实痛点：你不需要在每个平台单独配 bot，一个 Gateway 搞定所有。最近新增的 ACP（Agent Coding Protocol）和 Sub-agent 编排能力让它从”聊天助手”进化为”个人 AI 操作系统”。

技术亮点

  • 单进程多通道架构，20+ 消息平台开箱即用
  • 技能系统（Skills）实现模块化能力扩展
  • 支持 Cron 调度、Sub-agent 编排、Canvas 渲染
  • 本地运行，数据不离开你的设备

适用场景

想要一个真正属于自己的 AI 助手的个人开发者；需要跨平台统一 AI 入口的团队。

⭐ Stars: 291,398 | 📈 Today: +9,164 | Language: TypeScript

3. 666ghj/MiroFish

项目简介

群体智能预测引擎，通过构建数字平行世界中的多智能体社会模拟来预测未来走向。

为什么火？

这个项目的野心极大——上传一份种子材料（新闻、数据报告），它就构建一个”数字沙盘”，让成千上万个有独立人格和记忆的智能体自由演化，用”上帝视角”推演未来。概念新颖且实用，从政策模拟到小说结局推演都能玩。

技术亮点

  • 多智能体架构，每个 Agent 有独立人格、长期记忆和行为逻辑
  • 支持动态变量注入，可交互式调整模拟条件
  • 从种子信息自动构建高保真数字世界
  • 与同团队的 BettaFish（舆情分析）形成数据→分析→预测闭环

适用场景

政策研究者的预演实验室；商业决策者的沙盘推演工具；创意工作者的脑洞验证平台。

⭐ Stars: 11,387 | 📈 Today: +2,294 | Language: Python

4. karpathy/nanochat

项目简介

Karpathy 出品的极简 LLM 训练框架，号称”100 美元就能买到的最好 ChatGPT”。

为什么火？

这是 Karpathy 教育系列的最新力作。它把 LLM 全流程（tokenization、预训练、微调、评估、推理、聊天 UI）压缩到单 GPU 节点可运行的规模。GPT-2 级别的模型从 2019 年的 $43,000 训练成本降到了 $48（2小时 8×H100），最新优化甚至压到了 2 小时。

技术亮点

  • 单文件极简设计，代码可读性极强
  • --depth 一个参数控制模型规模，其他超参自动计算
  • 维护 GPT-2 Speedrun 排行榜，社区持续优化
  • 最新采用 NVIDIA ClimbMix 数据集，2.02 小时达到 GPT-2 水平

适用场景

想理解 LLM 训练全流程的学习者；需要快速实验小型语言模型的研究者。

⭐ Stars: 45,600 | 📈 Today: +355 | Language: Python

5. 666ghj/BettaFish

项目简介

多 Agent 舆情分析系统”微舆”，自动分析国内外 30+ 主流社交媒体和数百万条评论。

为什么火？

信息茧房是每个人的痛点。BettaFish 从 0 实现（不依赖任何框架），用多智能体协作的方式自动抓取、分析、生成研究报告。用户只需像聊天一样描述需求，系统全自动完成。与 MiroFish 组成”数据分析三板斧”。

技术亮点

  • 从 0 实现的多 Agent 架构，不依赖 LangChain 等框架
  • 覆盖 30+ 社交媒体平台的数据采集
  • 自然语言交互，聊天式提出分析需求
  • 自动生成结构化研究报告

适用场景

品牌公关团队；政策研究机构；市场分析师；关注舆论动态的个人用户。

⭐ Stars: 37,450 | 📈 Today: +514 | Language: Python

6. NousResearch/hermes-agent

项目简介

Nous Research 出品的自我进化 AI Agent，内置学习循环，能从经验中创建和改进技能。

为什么火？

这是目前少有的”会成长”的 Agent——它不只是执行任务，还会在使用中自动创建技能、改进技能、构建用户画像。支持 Telegram、Discord、Slack 等多平台，$5 VPS 就能跑。Nous Research 的品牌背书加持，加上 MIT 开源，吸引力很大。

技术亮点

  • 闭环学习：任务完成后自动提炼技能并持久化
  • FTS5 全文搜索实现跨会话记忆
  • Honcho 辩证式用户建模
  • 支持 200+ 模型，通过 OpenRouter 无锁定切换

适用场景

想要一个”越用越聪明”的个人 AI 助手的开发者；研究 Agent 自我改进机制的团队。

⭐ Stars: 3,034 | 📈 Today: +377 | Language: Python

7. pbakaus/impeccable

项目简介

一套面向 AI 编程助手的前端设计语言，让 AI 生成的 UI 不再千篇一律。

为什么火？

每个用过 AI 写前端的人都知道那个问题——Inter 字体、紫色渐变、嵌套卡片、灰色文字。Impeccable 用 7 个参考文件 + 17 个命令 + 反模式清单，系统性地纠正 AI 的设计偏见。它是 Anthropic 官方 frontend-design skill 的增强版。

技术亮点

  • 7 个领域参考文件：排版、配色、空间、动效、交互、响应式、UX 文案
  • 17 个操控命令（/audit, /critique, /polish, /animate 等）
  • 明确的”反模式”清单，告诉 AI 不该做什么
  • 即插即用，支持 Claude Code、Cursor 等主流 AI 编程工具

适用场景

用 AI 辅助前端开发的设计师和工程师；对 AI 生成 UI 质量不满意的团队。

⭐ Stars: 3,059 | 📈 Today: +1,288 | Language: JavaScript

8. msitarzewski/agency-agents

项目简介

一个精心设计的 AI Agent 人格库——每个 Agent 都是特定领域的专家，有独特个性和标准化交付物。

为什么火？

今日 +4,415 star，榜单增长最猛。它不是一个框架，而是一个”虚拟团队”——前端工程师、后端架构师、安全审计员、Reddit 社区忍者……每个 Agent 配有详细的身份设定、工作流程和交付标准。从 Reddit 帖子起家，迭代数月后成为最全的 AI Agent 人格库。

技术亮点

  • 支持 Claude Code、Cursor、Aider、Windsurf、Gemini CLI 等工具
  • 一键安装脚本，自动检测已安装工具
  • 每个 Agent 包含：身份/个性、核心工作流、代码示例、成功指标
  • 工程、产品、营销、运维等多部门覆盖

适用场景

想给 AI 编程助手赋予专业人格的开发者；组建”AI 虚拟团队”的技术团队。

⭐ Stars: 19,780 | 📈 Today: +4,415 | Language: Shell

9. alibaba/page-agent

项目简介

阿里巴巴开源的页内 GUI Agent，用自然语言控制网页界面，无需浏览器扩展或 Python。

为什么火？

与其他 Web Agent 不同，Page Agent 完全运行在页面内——纯 JavaScript，无需截图、OCR 或多模态模型。一行 script 标签就能集成到任何网页。对 SaaS 产品来说，这是最低成本的 AI Copilot 方案。

技术亮点

  • 纯文本 DOM 操作，不依赖截图或多模态 LLM
  • 一行代码集成，无需后端改造
  • 可选 Chrome 扩展支持多页面任务
  • 内置 Human-in-the-loop UI

适用场景

SaaS 产品快速集成 AI Copilot；企业 ERP/CRM 系统的智能表单填写；无障碍访问方案。

⭐ Stars: 2,692 | 📈 Today: +465 | Language: TypeScript

10. alirezarezvani/claude-skills

项目简介

170 个生产级技能/插件，覆盖 Claude Code、OpenAI Codex 和 OpenClaw，横跨工程、产品、营销、合规、C-level 咨询等领域。

为什么火？

AI 编程助手的”应用商店”来了。这个仓库把零散的 prompt engineering 打包成标准化的技能模块，每个技能包含指令文件、Python 工具和参考文档。一条命令安装，覆盖从安全审计到 Playwright 测试到自我改进 Agent 的方方面面。

技术亮点

  • 170 个技能覆盖 9 大领域（工程、产品、营销、合规、PM、C-level 等）
  • 支持 Claude Code 插件市场一键安装
  • 跨平台兼容：Claude Code、Codex、Gemini CLI、OpenClaw
  • SkillCheck 验证机制确保质量

适用场景

想快速提升 AI 编程助手能力上限的开发者；需要标准化 AI 工作流的企业团队。

⭐ Stars: 3,452 | 📈 Today: +259 | Language: Python

总结

今天的 Trending 画面非常清晰：Agent 生态的基础设施正在以惊人速度成熟。

几个值得关注的趋势：

  1. Agent 开始”长记忆”了：hermes-agent 和 openclaw 都在做跨会话记忆和自我改进，这是从工具到助手的关键跨越
  2. 中国开发者强势输出：MiroFish、BettaFish 来自同一团队，page-agent 来自阿里巴巴，中国 AI 开源力量不可忽视
  3. AI 设计质量成为焦点：Impeccable 的火爆说明大家已经受够了 AI 千篇一律的紫色渐变 UI
  4. 技能市场化：claude-skills 和 agency-agents 代表了 AI 能力模块化、可复用的大方向

如果只能关注三个项目，我推荐：MiroFish（最有想象力）、Impeccable（最实用）、nanochat（最有教育价值）。
