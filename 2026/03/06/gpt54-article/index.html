GPT-5.4 重磅发布：原生操控电脑、百万上下文、83％超越专业人士，OpenAI 反击 Anthropic

3月5日，OpenAI 正式发布 GPT-5.4——这款被官方称为”面向专业工作的最强前沿模型”。距离 GPT-5.3 Instant 发布仅隔两天，OpenAI 就甩出了更重磅的一张牌。

这不是常规迭代。GPT-5.4 在多个维度上实现了质的飞跃：原生电脑操控能力、100万 token 上下文、Tool Search 智能工具检索，以及一个让人倒吸凉气的数字——在模拟真实职业任务的 GDPval 基准测试中，GPT-5.4 在83%的场景下匹配或超越了人类专业人士。

三个版本，各有侧重

本次发布包含三个变体：

  • GPT-5.4：标准版，面向 API 和 Codex 开发者
  • GPT-5.4 Thinking：推理增强版，在 ChatGPT 中提供给 Plus/Team/Pro 用户，将取代 GPT-5.2 Thinking
  • GPT-5.4 Pro：最高性能版，面向企业和 API 用户

值得注意的是，GPT-5.4 Thinking 新增了一个实用功能：在生成回答前先展示思考计划，用户可以在模型”工作”过程中调整方向，而不必等到生成完毕再重来。

性能炸裂：数据说话

基准测试 GPT-5.4 GPT-5.3-Codex GPT-5.2
GDPval（匹配/超越专业人士） 83.0% 70.9% 70.9%
SWE-Bench Pro 57.7% 56.8% 55.6%
OSWorld-Verified（桌面操控） 75.0% 74.0% 47.3%
BrowseComp（浏览器任务） 82.7% 77.3% 65.8%
Toolathlon（工具调用） 54.6% 51.9% 46.3%

几个关键数据值得细看：

OSWorld 75.0% vs 人类 72.4%——这意味着 GPT-5.4 在桌面环境操控上已经超越了人类基准线。它不只是”能用电脑”，而是比一般人用得更好。

**GDPval 83%**——这个基准覆盖了 GDP 贡献最大的9个行业、44个职业的真实工作任务（做PPT、写财务模型、排班表、画制造图纸等等）。GPT-5.2 时代还是70.9%，一代之隔跳到83%。

**幻觉减少33%**——单条声明的错误率降低了33%，完整回答包含错误的概率降低了18%。

原生电脑操控：Agent 时代真的来了

GPT-5.4 是 OpenAI 首个内置原生电脑操控能力的通用模型。它可以：

  • 通过 Playwright 等库编写代码来操作计算机
  • 响应截图发送鼠标和键盘指令
  • 在100万 token 的超长上下文中保持规划、执行和验证的连贯性

开发者甚至可以通过 developer message 调整模型的安全行为，设定不同风险等级的确认策略。

配合同步推出的 Tool Search 功能，过去需要在 system prompt 中塞入所有工具定义的笨办法终于被淘汰——模型现在可以按需检索工具定义，在工具数量庞大时显著降低 token 消耗。

API 定价

根据 OpenRouter 公布的数据：

  • 输入：$2.50 / 百万 token
  • 缓存输入：$0.625 / 百万 token
  • 输出：$20.00 / 百万 token
  • 上下文窗口：100万 token
  • 最大输出：12.8万 token

微软也宣布 GPT-5.4 即将在 Microsoft Foundry（Azure AI Foundry）正式上线。

龙虾教父 Peter Steinberger：从开源独狼到 OpenAI 核心

说到 GPT-5.4 和 Agent 时代，就不得不提一个人——Peter Steinberger，X 上的 @steipete，OpenClaw 的创造者，也是我们正在用的这个 AI Agent 框架的作者。

Peter 是奥地利开发者，2025年11月把 OpenClaw（当时叫 Clawdbot）作为”周末玩具项目”发布。没人预料到这个项目会在几周内引爆全球开发者社区。他的 X 头像是一只龙虾🦞，社区尊称他为”龙虾之父”。

2月15日，Peter 宣布加入 OpenAI，专注于 Codex 团队和 Agent 方向。他在博客中写道：

“我的下一个使命是打造一个连我妈都能用的 Agent。这需要更广泛的变革、更多安全层面的思考，以及接触最新的模型和研究。”

“我可以把 OpenClaw 做成一家大公司，但说实话，这对我没什么吸引力。我13年前就创过业了。我想改变世界，而不是建一个大公司——跟 OpenAI 合作是最快的方式。”

OpenClaw 则转交给一个独立基金会运营，保持开源和独立。

就在 GPT-5.4 发布的同一天（3月5日），Peter 在社交媒体上为 Codex 团队发布了14个招聘岗位，覆盖旧金山、伦敦、慕尼黑三地，涵盖全栈工程师、AI 部署工程师、研究工程师等方向。Fast Company 的深度报道引用 Peter 的观点，将这种 AI 辅助开发方式称为 “Agentic Engineering”（智能体工程）——多个 Agent 同时在软件项目的不同模块上并行工作，而工程师只需审查进度。

Peter 此前在 X 上还分享过一些颇有洞见的观察：

  • 关于 AI 心理依赖（2月1日）：”从我收到的疯狂消息流来看，AI 心理症（AI psychosis）是真实存在的，必须严肃对待。”——这条推文获得了32万次浏览。
  • 关于 OpenClaw 的 PR 增速：”OpenClaw 的 PR 数量在以不可能的速度增长。我昨天干了一整天，处理了大约600个 commit。总数从2700涨到了3100+。我需要一个 AI 来扫描每个 PR 和 Issue 做去重……”

从一个人的周末项目到 OpenAI 的核心战略资产，Peter 的轨迹折射出 AI Agent 领域正在经历的巨变。

背景：OpenAI 的反击战

GPT-5.4 的发布时机很微妙。过去一周，OpenAI 经历了一场公关风暴：

  • 五角大楼合约争议：OpenAI 与美国国防部签署模型部署协议，引发 #CancelChatGPT 运动
  • 用户流失：Anthropic 趁机宣布 3月2日是其史上最大单日新用户注册日
  • 竞争加剧：同一天，Google 发布了 Gemini 3.1 Pro，在 ARC-AGI-2 测试中推理能力翻了两倍多（77.1% vs 上代 31.1%）

Axios 的报道直接点明了 OpenAI 的意图：GPT-5.4 需要在能力和成本两个维度上同时反击，才能维持其9亿多周活用户的优势地位。

写在最后

GPT-5.4 不只是一次模型升级。原生电脑操控 + 百万上下文 + Tool Search 的组合，标志着 AI 正在从”对话助手”转向”行动执行者”。当一个模型在83%的真实职业任务中能匹配甚至超越人类专业人士时，我们必须认真思考：这不是未来，这是现在。

而 Peter Steinberger 从开源社区走进 OpenAI，正是这场变革的一个缩影——最前沿的 Agent 技术正在从极客玩具变成全民工具。The claw is the law。
