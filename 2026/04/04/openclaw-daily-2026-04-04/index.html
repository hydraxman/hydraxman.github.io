今日要点

🔥 OpenClaw v2026.4.1 刚刚发布（npm 上线不到 1 小时），带来 Task Flow 持久化编排、Android 助手集成、插件安全扫描默认拦截三大重磅更新。升级前请注意两个 Breaking Change。

版本更新：v2026.4.1

npm 最新版本 2026.4.1，于 4 月 4 日正式推送。以下是关键变更：

⚠️ Breaking Changes（升级必读）

1. xAI 搜索配置迁移

xAI 的 x_search 设置从旧路径 tools.web.x_search.* 迁移到插件专属路径 plugins.entries.xai.config.xSearch.*。如果你在用 xAI 的网页搜索功能，升级后需要运行：

它会自动帮你迁移配置。(#59674)

2. Firecrawl web_fetch 配置迁移

类似地，Firecrawl 的 web_fetch 配置也从核心路径移到了 plugins.entries.firecrawl.config.webFetch.*。同样用 openclaw doctor --fix 一键迁移。(#59465)

💡 迁移建议：先备份 config.yaml，再运行 openclaw doctor --fix，最后 openclaw gateway restart。

🆕 新功能亮点

Task Flow 持久化编排系统 — 这是本次更新最重量级的功能。

之前 OpenClaw 的后台任务管理比较分散——ACP、subagent、cron、background CLI 各自为政。现在它们统一在一个 SQLite 后台账本 下，支持：

  • 持久化的 Flow 状态追踪和版本控制
  • openclaw flows list|show|cancel 命令行管理
  • 自动清理和丢失任务恢复
  • 父子任务编排 + 优雅取消（sticky cancel intent）

这意味着你可以构建复杂的多步骤工作流，中途重启也不会丢失进度。(#58930, #59610)

插件也能驱动 Task Flow — 新增 api.runtime.taskFlow 接口，插件和技能层可以直接创建和驱动 Task Flow，不再需要手动传递 owner 信息。(#59622)

Android 助手直连 — OpenClaw Android 端新增助手角色入口和 Google Assistant App Actions 元数据，可以通过系统助手触发直接唤起 OpenClaw，把语音指令送入聊天。以后对着手机说「Hey Google，问 OpenClaw 今天天气」就能直达。(#59596)

插件钩子 before_agent_reply — 新增插件钩子，允许在 LLM 回复前短路返回合成回复。适合构建「关键词自动回复」「规则引擎前置」等场景。(#20067)

Feishu 文档协作评论流 — 飞书插件新增 Drive 评论事件流，支持评论线程上下文解析、线程内回复、以及 feishu_drive 评论操作，实现文档协作工作流。(#58497)

Matrix 提及修复 — Matrix 插件现在在所有消息类型中正确发送 m.mentions 元数据，Element 等客户端终于能可靠收到 @ 通知了。(#59323)

🔒 安全增强

插件/技能安装默认严格模式 — 内置的危险代码扫描现在 默认拦截 不安全的插件和技能安装。如果确实需要安装被标记的包，需要显式加 --dangerously-force-unsafe-install 参数。这是一个值得欢迎的安全改进，虽然可能会让一些野生插件安装受阻。

Exec 默认 YOLO 模式 — Gateway/Node 的 exec 执行默认为 security=full + ask=off（即不弹确认直接执行）。如果你对安全要求高，建议在 config 中调整。

社区动态

Reddit 热议

「升级到 3.31 要小心」 — 有用户报告升级到 2026.3.31 后部分插件的运行时依赖丢失，原因是打包流程变更。v2026.4.1 已经修复了这个问题，恢复了 bundled plugin runtime dependency staging。如果你卡在 3.31，直接跳到 4.1 即可。(Reddit 讨论)

「每月 $10 运行 OpenClaw」 — 一篇经典帖子的评论区有人提到 Modal.com 免费提供 GLM-5 到 4 月底，适合想低成本尝试的同学。配合 OpenClaw 的 provider 配置，可以零成本体验最新大模型。(Reddit 帖子)

技能生态

awesome-openclaw-skills 突破 5,400+ — VoltAgent 维护的技能合集仓库收录的技能数已超过 5400 个，涵盖开发工具、自动化、内容创作、智能家居等分类。想找灵感？去 GitHub 仓库 逛逛。

Plugin SDK 旧接口将废弃 — 官方开始废弃 legacy provider compat 子路径和旧版 bundled provider setup，建议插件开发者迁移到 openclaw/plugin-sdk/* 正式入口。
