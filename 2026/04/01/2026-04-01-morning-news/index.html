🌍 国际要闻

1. 美以对伊作战进入第33天，Trump称”2-3周内结束”

美以联军对伊朗的军事行动已持续超过一个月。Trump向盟友表示”自己找石油去”，暗示美方即将撤离。教皇利奥敦促Trump结束战争。欧盟警告油气价格短期内无法恢复正常水平。伊朗批准法案对非交战国船只通过霍尔木兹海峡收费，目前已有中国船舶通过。

2. 伊朗在战争中使用儿童担任安保角色

BBC报道及目击者证实，伊朗在战时征调未成年人参与安全任务，引发严重的国际人权关切。伊朗家庭因战争立场分裂，”你不再是我姐妹”的冲突在家庭间蔓延。

3. 以色列空袭贝鲁特，宣布战后继续控制黎巴嫩南部

救援人员在废墟中搜寻遇难者。联合国援助负责人警告这意味着新一轮军事占领。联合国维和部队在黎巴嫩遭路边炸弹袭击，已有人员伤亡。

4. NASA Artemis II 登月任务准备就绪

数十年来首次载人登月发射即将进行，火箭状态和天气条件均已达标。Paul McCartney今晚将在Apple Park为苹果50周年庆典献唱——太空与音乐，同一天的浪漫。

5. 印度启动超14亿人口大规模人口普查

这是印度有史以来最大规模的人口统计行动，数据将影响未来十年的政策制定和资源分配。

6. 意大利连续第三届无缘世界杯

四届世界杯冠军意大利在点球大战中输给波黑，连续三届无缘世界杯。波黑球迷举国欢庆。

🌐 科技新闻

1. Claude Code 源码泄露：51.2万行代码曝光

Anthropic 旗舰 CLI 工具 Claude Code 的完整源码通过 npm source map 意外泄露。泄露内容包括”假工具”机制、”挫败感正则表达式”、隐身模式，甚至还有一个 Tamagotchi 风格的”宠物”彩蛋和 always-on agent 功能。竞争对手和开发者社区预计将花数周时间研究这些代码。

2. 量子计算破解加密的时间大幅提前

Ars Technica 报道，量子计算破解关键加密所需的资源远少于此前预期。Google 将”Q Day”（量子计算攻破 RSA 加密的日子）时间线从 2030 年代提前至 2029 年，这将加速全球向后量子密码学的迁移。

3. OpenAI 以 $852B 估值完成融资

OpenAI 尚未上市便以 8520 亿美元估值完成新一轮融资，其中散户投资者投入 30 亿美元，总融资规模达 1220 亿美元。

4. 中国无人驾驶出租车大面积宕机

Wired 报道，中国 robotaxi 出现大规模故障，导致乘客被困高速公路。robotaxi 公司拒绝透露自动驾驶车辆需要远程协助的频率。Tesla 也承认其 robotaxi 有时由远程人类驾驶。

5. 伊朗威胁4月1日起攻击美国科技巨头

伊朗政府宣布将针对苹果、谷歌、微软等18家美国科技公司在西亚的基础设施发动攻击。

6. 自传播恶意软件毒害开源项目并清除伊朗机器数据

Ars Technica 报道了一种新型自传播恶意软件，它污染开源软件供应链，同时专门针对伊朗的机器进行数据擦除。

7. Oracle 大规模裁员，涉及销售、工程、安全部门

科技巨头 Oracle 进行了”重大”裁员，影响数千名员工。

8. Apple 测试 Siri 多指令处理功能

Bloomberg 报道，苹果正在测试让 Siri 一次处理多个命令的新功能。国行苹果 AI 也终于上线。

9. Google 允许美国用户更改 Gmail 地址

无需删除账户即可更改邮箱用户名，这是 Gmail 历史性的功能更新。

10. 微软经历2008年以来最差季度表现

受 AI 投入回报质疑影响，微软市值在第一季度蒸发近四分之一。但昨夜美股全线反弹。
