一句话概括： Hermes Agent 是 Nous Research 开源的全平台 AI Agent 框架——终端里能干活，微信/Telegram/Discord 里能聊天，还能定时执行任务、自我学习。这篇文章手把手教你从零安装到全平台部署。

📖 Hermes Agent 是什么？

如果你用过 Claude Code、OpenAI Codex CLI 或者 Cursor，那 Hermes Agent 就是同一类东西——但是开源的、全平台的、可自我进化的。

它的核心能力：

  • 🧠 持久记忆 —— 跨会话记住你的偏好、项目结构、踩过的坑
  • 🛠 工具调用 —— 终端执行命令、读写文件、搜索网页、浏览器自动化、MCP 服务器对接
  • 💬 全平台网关 —— 同一个 Agent 同时跑在微信、Telegram、Discord、Slack、WhatsApp、飞书等 10+ 平台
  • 📚 技能系统 —— 解决了复杂问题？自动保存为 Skill，下次直接用
  • ⏰ 定时任务 —— Cron 调度，定时抓新闻、发日报、同步代码……
  • 🔄 模型随便换 —— 支持 18+ 模型提供商，OpenRouter、Anthropic、OpenAI、DeepSeek、本地模型统统能用

🏗 整体架构一览

先看全局架构图，心里有数再动手：

核心设计思路是 **”一个 Agent 核心，多种接入方式”**：

  • CLI 模式：终端里直接对话，程序员日常最常用
  • Gateway 模式：作为服务运行，接入各种聊天平台
  • Agent Core：统一的 AIAgent 循环，负责 LLM 调用、工具分发、记忆管理

⚡ 安装：一行命令搞定

前置条件

  • macOS / Linux / WSL（Windows 原生暂不支持）
  • Python 3.10+（推荐 3.11）
  • 一个 LLM API Key（OpenRouter、Anthropic、OpenAI 等任选其一）

一键安装

安装脚本会自动：

  1. 克隆仓库到 ~/.hermes/hermes-agent/
  2. 创建 Python 虚拟环境
  3. 安装所有依赖
  4. 把 hermes 命令加到你的 PATH

🧠 记忆与技能系统

持久记忆

Hermes 自动维护两种记忆：

  • 用户画像（User Profile） —— 记住你是谁、你的偏好、沟通风格
  • 工作笔记（Memory） —— 环境信息、项目约定、踩坑经验

记忆跨会话持久存在，不需要每次重复告诉 Agent 你的情况。

🎯 总结

Hermes Agent 是目前开源 AI Agent 框架中最全面的方案之一：

  1. 入门简单 —— 一行命令安装，几分钟配置好就能用
  2. 功能强大 —— 终端、文件、网页、浏览器、MCP……工具生态完整
  3. 全平台部署 —— 微信、Telegram、Discord 等 10+ 平台一键接入
  4. 可自我进化 —— 记忆 + 技能系统让 Agent 越用越聪明
  5. 完全可控 —— 开源、本地部署、数据在你手里

如果你正在找一个能真正”干活”的 AI Agent，而不只是聊天机器人，Hermes Agent 值得一试。
