今日要点

OpenClaw 连发两个版本（2026.4.8 Changes + Hotfix），带来 Memory Wiki 全栈回归、全新 openclaw infer 推理 CLI、可插拔压缩引擎，以及大量渠道和安全修复。社区热议 2026.4.5 带来的 AI Agent 生态成熟度跃升。

🚀 版本更新：2026.4.8

4 月 8 日一天之内推了两波更新，先是 Changes 大版本，随后紧跟 Hotfix 修补打包问题。以下是最值得关注的变化：

一、openclaw infer — 一站式推理中心

新增一等公民 CLI 命令 openclaw infer，统一调用各 Provider 的推理能力，覆盖：

  • 文本推理 — 直接调模型
  • 媒体生成 — 图片 / 音乐 / 视频，自动在 Provider 间 fallback
  • Web 搜索 — 利用已配置的搜索 Provider
  • Embedding — 向量化任务

这意味着你不再需要记住每个 Provider 的 API 差异，openclaw infer 帮你屏蔽底层复杂度。

实战提示：在脚本中用 openclaw infer image "一只赛博朋克风格的龙" 就能直接生成图片，自动选最优 Provider。

二、Memory Wiki 全栈回归

由社区贡献者 @vincentkoc 主导，Memory Wiki 系统完整回归，包括：

  • 插件 + CLI + 同步/查询/应用工具链
  • 结构化 claim/evidence 字段
  • 编译摘要检索、claim 健康度 lint
  • 矛盾聚类、过时性仪表盘
  • 新鲜度加权搜索 — 越新的记忆权重越高

这是 OpenClaw 向”真正有长期记忆的 AI Agent”迈出的关键一步。配合 Dreaming 系统（会话转录自动注入梦境语料库），Agent 的记忆深度前所未有。

三、可插拔压缩引擎

PR #56224 引入 agents.defaults.compaction.provider 配置，支持自定义压缩 Provider。默认仍然是 LLM 摘要，但你现在可以：

  • 插入自己的压缩算法（比如基于 embedding 相似度的去重）
  • Provider 失败时自动回退到 LLM 摘要

加上新增的 压缩检查点持久化 和 Sessions UI 的分支/恢复操作（#62146），运维人员终于可以检查和恢复压缩前的会话状态了。

四、Webhook 入站插件

#61892 新增内置 Webhook 插件，外部自动化系统可以通过带 shared-secret 的 HTTP 端点创建和驱动 TaskFlow。这对于把 OpenClaw 集成进 CI/CD 或企业工作流至关重要。

五、新 Provider 支持

Provider 亮点
Google Gemma 4 原生路由 + thinking-off 语义保留 (#61507, #62127)
Arcee AI Trinity 模型目录 + OpenRouter 支持 (#62068)
Ollama 视觉 自动检测 vision 能力，支持图片附件 (#62193)
Anthropic Claude CLI 重新成为本地 Anthropic 首选路径

六、系统提示实验支持

新增 agents.defaults.systemPromptOverride 配置，方便进行受控的 prompt 实验。同时 Heartbeat 提示段落可以独立控制——不再每轮都注入 Heartbeat 指令。

🔧 重要修复

  • 全渠道打包修复：Telegram、BlueBubbles、Feishu、Google Chat、IRC、Matrix、Mattermost、Teams、Nextcloud Talk、Slack、Zalo 的 npm 安装不再找不到 dist/extensions/*/src/* 文件
  • Slack 代理支持：Socket Mode WebSocket 现在遵守 HTTP(S) 代理设置 (#62878)
  • DNS 安全：受信代理模式下跳过 DNS pinning，代理沙箱正常工作 (#59007)
  • iOS 连接体验：结构化错误 UI 替代字符串匹配，配对/认证失败信息更明确 (#62650)
  • Control UI：Webchat 支持 /tts 音频播放，修复移动端 exec 审批 UI 遮挡问题

🌍 社区动态

Reddit 热帖：2026.4.5 讨论

Reddit r/openclaw 社区对 2026.4.5 版本反响热烈，多位用户表示”开源社区的力量让 OpenClaw 的增长潜力巨大”。核心讨论点：

  • 内置视频/音乐生成 让 Agent 的多模态能力直线上升
  • Claude CLI 回归 为本地 Anthropic 用户扫清障碍
  • 媒体 Provider 自动 fallback 大幅降低配置心智负担

第三方分析

据 openclaw.com.au 分析，OpenClaw 正在从”一堆 hack 拼起来的 Agent Shell”蜕变为真正的 Agent 操作系统——Marketplace 安装、插件发现、Provider 注册表、更新路径都在走向产品化。

值得关注的未发布功能：可插拔沙箱后端、GitHub main 分支安装支持、Firecrawl 搜索/抓取工具、快速 /btw 侧问流程、更紧密的健康监控。
