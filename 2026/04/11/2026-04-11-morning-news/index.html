🌍 国际要闻

1. 美伊停火谈判在巴基斯坦正式启动

美国谈判代表抵达巴基斯坦，与伊朗展开停火谈判。各方已同意以伊方提出的十点方案为谈判基础，这是近期中东紧张局势中的重要外交进展。

2. 匈牙利大选进入最后冲刺

匈牙利反对派候选人在民调中大幅缩小差距，有望终结欧尔班长期执政。选民正在进行最后投票，结果可能改变欧洲政治格局。

3. 以色列空袭黎巴嫩政府大楼致多名军官死亡

以军袭击导致黎巴嫩军方人员伤亡，真主党随后发射火箭弹回击，损毁一座有1500年历史的以色列教堂。

4. 约旦河西岸定居者暴力事件升级

巴勒斯坦人在以色列定居者袭击中被射杀，暴力事件持续升级。

5. 南美贸易摩擦加剧

哥伦比亚宣布对厄瓜多尔商品加征100%关税，回应厄瓜多尔此前的关税提高措施。巴西则与美国达成合作协议，联合打击武器和毒品走私。

🌐 科技新闻

1. Artemis II 成功绕月返回地球

NASA Artemis II 任务历史性完成载人绕月飞行并安全溅落返回，标志着人类重返月球计划的重大里程碑。这是自阿波罗时代以来人类首次飞越月球。

2. Anthropic 临时封禁 OpenClaw 创始人访问 Claude

Anthropic 因安全考虑暂时限制了 OpenClaw 创始人的 Claude API 访问权限，引发开源社区对 AI 平台治理的讨论。

3. 20岁男子因向 Sam Altman 住宅投掷燃烧瓶被捕

OpenAI CEO Sam Altman 的住所遭到燃烧瓶袭击，嫌疑人已被警方逮捕。Altman 随后发表声明回应。

4. 法国启动政府 Linux 桌面计划

法国政府宣布将公共部门电脑从 Windows 迁移至 Linux，以减少对美国科技的依赖。这是欧洲数字主权运动的重要一步。

5. 特斯拉连续第8周下跌

特斯拉股价持续走弱，摩根大通警告可能再跌60%。华尔街分析师对其前景表达悲观预期。

6. Microsoft 开始移除 Windows 11 中的 Copilot 品牌

微软开始从 Windows 11 应用中去除 Copilot 品牌标识，可能预示着 AI 助手战略调整。

7. FBI 从 iPhone 通知数据中提取已删除的 Signal 消息

FBI 通过 iPhone 通知缓存成功提取嫌疑人已删除的 Signal 加密消息，引发隐私安全讨论。
