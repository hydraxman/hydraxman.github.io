从秧BOT到功夫BOT：宇树机器人如何用一支秧歌舞征服全球，引爆千亿赛道

2025年1月28日，蛇年除夕夜。

当16台身高1.8米、身穿红白花棉袄的人形机器人走上央视春晚舞台，与人类舞者一起扭起东北秧歌、空中抛接手绢时——全球互联网炸了。

X（Twitter）播放量突破480万次。YouTube十万+。Reddit多个板块同时霸榜。从路透社到南华早报，从印度媒体到台湾联合报，从硅谷工程师到欧洲网友，所有人都在问同一个问题：

这些中国机器人，到底是怎么做到的？

这些机器人来自杭州的宇树科技（Unitree Robotics），型号H1「伏羲」。节目名叫《秧BOT》——一个绝妙的双关：秧歌+Robot。导演是张艺谋，没错，就是2008北京奥运开幕式的那个张艺谋。

🌍 外媒集体震动：从惊叹到焦虑

路透社：一个”不太可能的场所”

路透社用了一个很精妙的措辞——春晚是展示机器人技术的”an unlikely venue“（一个不太可能的场所），但效果出奇地好。16台全尺寸人形机器人与人类舞者同台，”让数百万观众发出了震惊的赞叹”。

路透社特别指出，春晚后数周，宇树科技创始人王兴兴获安排与中国国家主席习近平会面——这个细节被台湾《中时新闻网》等媒体反复引用，视为官方对机器人产业高度重视的信号。

南华早报：阿里云加持的”舞王”

《南华早报》深挖了技术细节：H1搭载阿里云技术基础设施，实时处理高频空间数据，实现精确定位和协调运动。AI算法让机器人”理解”音乐并执行类人舞蹈动作。

报道称这在中美科技竞争中为中国”增添了一个筹码”。

美国Business Today：直接拿来跟DeepSeek并列

“随着DeepSeek证明高性能AI可以以极低成本构建，Unitree展示了能够精准起舞的人形机器人，中国正在挑战美国的传统科技领导地位。”

西班牙Euro Weekly News：嘲讽马斯克的Optimus

欧洲媒体毫不客气：

“伊隆·马斯克的Optimus机器人还在小心翼翼地走路，仿佛害怕绊倒。与此同时，AI驱动的H1机器人与人类舞者灵活、安全地同步起舞。”

印度Times of India：为什么这支舞让人不安？

《印度时报》专门写了一篇分析文章——标题就叫”Robots dance with humans in China in a viral event: Why we find it disturbing”（机器人在中国与人类共舞的病毒式传播事件：为什么我们觉得不安）。文章深入探讨了”恐怖谷效应”，认为机器人流畅但无表情的动作触发了人类本能的不适感。

New Atlas：不仅跳舞，还能转手绢

澳大利亚科技媒体New Atlas特别强调了一个细节：

“16台H1机器人不仅同步跳舞，每一台都完美地在空中旋转并接住了手绢。”

这个动作的难度在于精细的手部控制和空间感知——对机器人来说，这比跳舞本身更难。

🇹🇼 台湾媒体的视角：从惊叹到焦虑

台湾媒体对秧BOT的报道非常密集，角度也独特——既有技术惊叹，也有地缘政治焦虑。

联合报：春晚后机器人变”打工人”

台湾《联合报》/《世界日报》多次追踪报道。最让人印象深刻的是这篇——《在春晚跳舞暴紅的機器人，開始全陸打工》：

春晚后，宇树机器人一机难求，催生了全中国的机器人租赁热潮：

  • 日租金最高1.5万元人民币（约2067美元）
  • 有商家出租十余次，收入超20万元，已回本
  • 租赁档期排到下半年
  • 商场、酒店、企业年会、学校……到处能看到穿着花棉袄的机器人

联合报还报道了宇树的后续进化：春晚两周后发布**”功夫BOT”**——G1机器人展示中国功夫，回旋踢、扫堂腿样样精通。宇树宣称算法升级到了”任意动作任意学“的程度。

中时新闻网：美专家紧盯2026春晚

2026年春晚前夕，台湾《中时新闻网》以”美專家緊盯2026春晚——陸AI+製造戰略核心：人形機器人“为标题发文，指出：

春晚已经成为观察中国科技战略的窗口。去年16台宇树机器人的秧歌表演引发广泛讨论，数周后，该公司创始人更获安排与习近平会面，显示官方对机器人产业的高度重视。

联合报/经济日报：机器人企业砸亿元抢春晚

台媒跟踪报道了2026春晚的赞助竞标战：

  • 智元机器人开价6000万元
  • 宇树直接加码到1亿元
  • 最终宇树第三度登台，银河通用、魔法原子也加入

台湾《经济日报》评论称，这标志着春晚正从”展示酷炫技术”转向”呈现中国前沿产业核心技术实力”。

联合报：马斯克都点赞了

2025年12月，6台宇树G1机器人在王力宏演唱会上伴舞、做后空翻，马斯克在X上转发点赞。台湾联合报以”馬斯克點讚！人形機器人登王力宏演唱會“为标题报道，引用网友评论：

“几个月前在春晚舞台上还像刚学走路的小孩，如今已经成为了专业的舞者。”

联合报：机器人租赁也”内卷”了

到2025年底，台媒又追踪到新的变化——机器人租赁价格暴跌九成：

“从年初日租破万元、一机难求，到如今3000元人民币即可租到机器人加机器狗一天。”

台湾媒体精准捕捉到了一个中国式商业故事的完整弧线：爆红→疯抢→内卷→价格崩盘。

它告诉世界：中国不仅能造出先进的机器人，还能让它们穿着花棉袄扭秧歌、甩手绢——而且是在全球收视率最高的电视节目上，由张艺谋来导演。

Aspect-Oriented Programming in Android

What is AOP?​

In computing, aspect-oriented programming (AOP) is a programming paradigm that aims to increase modularity by allowing the separation of cross-cutting concerns. It does so by adding additional behavior to existing code (an advice) without modifying the code itself, instead separately specifying which code is modified via a “pointcut” specification, such as “log all function calls when the function’s name begins with ‘set’”. This allows behaviors that are not central to the business logic (such as logging) to be added to a program without cluttering the code, core to the functionality. AOP forms a basis for aspect-oriented software development.
