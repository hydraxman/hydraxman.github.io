领导力21法则读书笔记

追随这些法则，人们就会追随你。

The 21 irrefutable laws of leadership, follow them and people will follow you.

书评

优点:

很多观点切中要害，非常实际、实用和恳切。启发性很强。理论和概念的拔升和抽象到了一定的高度，很有水平。

作者对事物有很强的概括和梳理能力。对领导力的思索很深刻。

有很多例子和故事来辅助理解。例子都很贴切，也很有趣。

缺点

有些观点对于领导者太卷了，容易把人吓跑。21条，数量太多了，容易让人抓不住重点。有些观点不够与时俱进，笔者似乎对于科技的进步不够重视。有些说法比较绝对。

书籍脉络

  1. 盖子 Lid
    1. 个人和组织的办事效力与领导力成正比
      1. 人际交往能力
      2. 规划能力和战略思维
      3. 远见
      4. 业绩
    2. 麦当劳例子
    3. 接管和改善一个组织：培训所有员工，解雇原来的领导者
  2. 影响力 Influence
    1. 如果没有影响力，你永远不可能领导别人
    2. 特蕾莎例子：真正的领导者发言时人们会洗耳恭听
    3. 关于领导力的五大误解
      1. 管理者就是领导者
      2. 企业家就是领导者
      3. 知识渊博的人就是领导者
      4. 引领潮流的人就是领导者
      5. 位高权重的人就是领导者
    4. 衡量领导力的尺度只能是影响力
      1. 知识
      2. 直觉
      3. 经验
      4. 过去的成就
      5. 能力
  3. 过程 Process
    1. 领导力的提升需要日积月累，而非一日之功
    2. 领导者都是善于学习之人
      1. 不知道你不知道
      2. 知道自己需要知道
      3. 知道自己不知道
      4. 我知道，我成长，我发现
      5. 因为我知道
        1. 阶段五：你拥有高度的直觉
    3. 实干家罗斯福：做了673场演讲，全国各地旅行20000英里
  4. 导航 Navigation
    1. 任何人都可以掌舵，唯有领导者才能设定航线
      1. 看得比别人多，看得比别人远，在别人看到之前看到
    2. 过去的经验
    3. 做承诺前或考察各种情况
    4. 多听从他人意见
    5. PLAN AHEAD
      1. Predetermine a course of action
      2. Lay out your goals
      3. Adjust your priorities
      4. Notify key personnel
      5. Allow time for acceptance
      6. Head into action
      7. Explore problems
      8. Always point to the success
      9. Daily review your plan
    6. 一个成功方案的最大阻碍
      1. 对改革的恐惧、未知
      2. 对未来的不确定
      3. 想象力的缺乏
  5. 增值 Addition
    1. 提升价值，增加利润
    2. 辛内加尔领导Costco的例子
      1. 如果想独领风骚，那你就得不到员工的忠心
      2. 如果不能把功能让给别人（把失败的责任留给自己），那你就无法激励别人，最终只能自毁前程
    3. 拥有正确的领导动机
      1. 领导者要服务他人，提升他人的价值
      2. 你让你的追随者改善生活了吗？
  6. 根基 Solid Ground
    1. 信任是领导力的根基所在
    2. 品格
      1. 彰显一贯性、潜力和赢得尊重
  7. 尊重 Respect
    1. 人们愿意追随比自己强的领导者
    2. 一个人的领导力越强，就能越快发现别人的领导力潜质，或者发现别人领导力的不足
    3. 获得尊敬的条件
      1. 领导能力
      2. 对他人的尊重
      3. 勇气
      4. 过去的佳绩
      5. 忠诚度
      6. 为他人增加价值
  8. 直觉 Intuition
    1. 每个人都在自己擅长的领域富有直觉
    2. 妻子玛格丽特对着装的直觉
  9. 吸引力 Magnetism
    1. 你只能吸引和你相似的人
      1. 你所吸引的人不是由你的愿望决定的，而是由你的为人决定的
      2. 如果你觉得你所吸引的人应该更优秀，那就到了你应该提高自己能力的时候了
  10. 亲和力 Connection
    1. 得人心者得天下
    2. 与人交往方面，感情要先于理智
    3. 我们都希望演讲者的讲话更有人情味
    4. 与个人亲善
      1. 展现真我
        1. 我的个性是什么：优点和缺点？
        2. 能力？最大资本和不足？
      2. 坦率真诚
      3. 了解对象
      4. 身体力行
      5. 身临其境
      6. 关注他们，而不是你自己
      7. 信任他人
      8. 指明方向，带来希望
    5. 在着手工作之前，先联络感情： 优秀的领导都是人际关系大师
  11. 核心圈 Inner Circle
    1. 一个领导者的潜力，由最接近他的人决定
    2. 将谁划入核心圈
      1. 是否对其他人有很大的影响力
      2. 是否能给团队带来互补性的才能
      3. 在团队里是否担任要职
      4. 能否增加我团队的价值
      5. 给核心圈的其他成员带来积极的影响
    3. 鉴别 - 培养 - 吸收
    4. 不断提高你的核心圈素质
    5. 当你的直属人数超过7个人，不是每个人都能直接领导的时候，应该建立核心圈
  12. 授权 Empowerment
    1. 有安全感的领导者才会授权予人（例子：亨利福特，反例，不授权，坚持T型车，导致持续亏损，直至亨利二世接管）
    2. 谁才是老板？授权的障碍：
      1. 渴望工作上的安全感
      2. 抗拒改变
      3. 缺乏自我肯定
    3. 找到有能力的领导者并授权给他们
  13. 镜像 Picture
    1. 看到别人怎么做，大家也会怎么做（兄弟连电视剧对美国军队的影响）
    2. 领导者必须要成为梦想家，你的领导水平越高，你的工作就越多是关于预见未来可多事情
    3. 一定要实际：对愿景的有效示范是把镜像变成现实
    4. 领导者愿景的典范
      1. 下属总是在观察你的所作所为
      2. 教正确的事总比坐正确的事容易
      3. 领导者能给下属的最宝贵的东西就是一个好的榜样
  14. 接纳 Buy-In
    1. 人们先接纳领导者，然后接纳他描绘的愿景
    2. 领导者先于愿景：你就是信息
    3. 领导者和愿景：并非两者之中择一
      1. 当追随者不喜欢领导者及其愿景时，他会去找另一个领导者
      2. 当追随者喜欢领导者，却不喜欢其愿景时，他们愿意改变愿景
      3. 当追随者同时欣赏领导者及其目标时，他们会全力以赴
    4. 成功是以你是否有能力带领下属达到他们认同的目标来衡量的
  15. 制胜 Victory
    1. 领导者为他的团队找出一条制胜之路
    2. 丘吉尔绝不接受任何妥协
      1. 我们的目标是什么？用一句话回答，赢得胜利！不惜任何代价，不管面对多少恐怖，我们就是要打胜仗。不管这条路有多长、多难。不能获胜，就只有灭亡。
    3. 胜利的要素
      1. 统一的目标
      2. 多样化的技能
      3. 一位愿为胜利而献身、致力于发挥队员潜能的领导者
    4. 制胜是领导者的本分
  16. 动势 Big Mo
    1. 动势是领导者最好的朋友
    2. 皮克斯和玩具总动员，对比同时间迪士尼
    3. 当对方连续得分、动势越趋强大的时候，优秀的教练会立刻叫暂停。
    4. 一旦产生了动势，你就不会担心小问题。
    5. 如果没有了动势，就连最简单的任务也会变成登天难题。
      1. 动势是放大器
      2. 动势可以美化领导者
      3. 动势帮助下属们表现得更好
      4. 掌握动势的方向要比创造动势容易
      5. 动势是带动改变最有力的因素
      6. 创造动势是领导者的责任
      7. 动势从领导者自身产生
    6. 激励是创造动势的关键要素。
  17. 优先次序 Priorites
    1. 忙碌不一定等于成效
    2. 三R法则
      1. Requirement 什么是你必须做的分内之事？
      2. Return 什么能带来最大的收益？
      3. Reward 什么能带来最大的回报？
    3. 在所有活动中，只要集中注意最重要的那20%，你就能获得你付出努力80%的回报。
      1. 领导 Leadership
      2. 沟通 Communicating
      3. 创作 Creating
      4. 联络 Networking
    4. 杰克韦尔奇 快刀斩乱麻，集中力量：保留能占领世界第一或第二的生产线
  18. 舍得 Sacrifice
    1. 领导必须先舍后得
    2. 牺牲是领导力的精髓：成功的领导者懂得牺牲次要的东西
    3. 地位越高，舍弃越多
    4. 牺牲是一个持续的过程
  19. 时机 Timing
    1. 掌握时机与善用策略同样重要
    2. 时机决定一切
      1. 在错误的时机采取错误的行动，结果是酿成灾难
      2. 在错误的时机采取正确的行动，结果是抵制
        1. 认识：对局势有清楚的认识
        2. 成熟
        3. 信心：人们往往追随心里有谱的领导者
        4. 果断
        5. 经验
        6. 直觉
        7. 准备
      3. 在正确的时机采取错误的行动，结果是犯错
      4. 在正确的时机采取正确的形成，结果是成功
    3. 行动的目标？谁负责？市场、行业如何
  20. 爆炸性倍增 Explosive Growth
    1. 培养追随者，得到相加的效果；培养领导者，得到倍增的效果。
    2. 帮助别人领导
      1. 培养追随者的领导者：
        1. 发展最下层的20%；
        2. 专注于人的弱点；
        3. 一视同仁；
        4. 在一般人身上花费时间；
        5. 以加法方式成长
        6. 只影响到身边接触到的人。
      2. 培养领导者的领导者：
        1. 发展最优秀的20%。
        2. 专注于人的长处；
        3. 因人而异；
        4. 在人才上投资时间；
        5. 以乘法方式成长。
        6. 影响力扩及甚远。
    3. 培养领导者的挑战：领导者难觅、难聚、难留
      1. 如果你一直为你手下的领导者提升价值，他们就会愿意一直追随你。
    4. 阶段：提高自己》发展团队》培养领导者
  21. 传承 Legacy
    1. 一个领导者的长久价值由其继承者决定
    2. 发展你的领导遗产
      1. 清楚自己希望留下的遗产
      2. 实现自己希望留下的遗产
      3. 挑选你事业的继承人
      4. 确保顺利传递交接棒
