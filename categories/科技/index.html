老外看呆了：2025春晚机器人扭秧歌，Unitree宇树科技如何震撼世界

2025年1月28日，农历蛇年除夕之夜。当16台身高1.8米、身穿红白旗袍马甲的人形机器人走上春晚舞台，与人类舞者一起扭起东北秧歌、抛接手绢时，不仅14亿中国观众看呆了——这段视频迅速传遍全球社交媒体，在Twitter/X上的播放量突破480万次，YouTube上的播放量也迅速超过10万。

这些机器人来自杭州的宇树科技（Unitree Robotics），型号为H1「伏羲」。这场表演由张艺谋导演——没错，就是那位执导了2008年北京奥运会开幕式的张艺谋。

而海外的反应，可以用两个字概括：炸了。

外媒怎么说：从惊叹到焦虑

路透社：中国机器人公司准备好了

路透社在报道中指出，春晚已经成为中国机器人企业展示技术实力的重要舞台——“一个不太可能的场所”（an unlikely venue），却产生了巨大的影响力。16台Unitree全尺寸人形机器人与人类舞者同台演出，”让数百万观众发出了震惊的赞叹”。

南华早报：阿里云赋能的「舞王」

《南华早报》详细报道了技术细节：这些H1机器人搭载了阿里云提供的技术基础设施，能够实时处理高频空间数据，实现精确的定位和协调运动。Unitree的AI算法让机器人能够”理解”音乐并执行类人的舞蹈动作。报道指出，这一里程碑式的表演反映了中国机器人产业的进步，在中美科技竞争中为中国增添了一个筹码。

美国媒体Business Today：中国正在挑战传统科技领袖

美国媒体Business Today直接将此事与DeepSeek并列讨论：

“随着DeepSeek证明高性能AI可以以极低成本构建，Unitree展示了能够精准起舞的人形机器人，中国正在挑战美国的传统科技领导地位。”

西班牙Euro Weekly News：对比马斯克的Optimus

欧洲媒体则毫不客气地拿来做对比：

“现在，美国和欧洲正密切关注中国及其不断演进的科技突破——DeepSeek在AI上碾压了世界其他玩家，而伊隆·马斯克的Optimus机器人还在小心翼翼地走路，仿佛害怕绊倒。与此同时，AI驱动的H1机器人与人类舞者灵活、安全地同步起舞。”

印度媒体India Today：对美国市场的威胁

印度媒体India Today认为，DeepSeek和H1机器人的接连亮相，展示了”中国在全球AI和机器人领域日益增长的存在感”，并称这”可能对美国市场构成威胁”。

Business Insider：关注军事应用

当然，也有美国媒体表达了担忧。Business Insider在报道中特别指出，除了春晚上的H1舞蹈机器人，Unitree还制造了四足机器狗，这些机器人在2024年曾出现在中柬联合军事训练演习中。不过报道也承认，美国军方同样使用了波士顿动力的非武装机器人。

海外网友的真实反应：从”太酷了”到”有点瘆人”

社交媒体上的反应远比媒体报道更加多元和有趣。

惊叹派

美国记者Ben Norton在X上发布了机器人舞蹈视频，写道：

“中国现在有了会跳舞的机器人——除了能以更低成本生产比美国科技巨头更好的AI，而且还是开源的。这就是美国对中国发动经济战和科技战的原因——美国不希望自己的垄断地位受到任何竞争。”

还有X用户调侃：

“马斯克的机器人还在学走路的时候，中国的机器人已经像专业舞者一样跳起来了。”（While Musk’s robot is still learning to walk, China’s robots are already dancing like pros.）

质疑派

Reddit上的r/robotics板块则出现了冷静的技术讨论。有用户泼冷水：

“恭喜，你可以给同一型号的一组机器人在同步时钟上输入相同的指令集。没什么新东西，没什么了不起的。”（grats, you can feed a set of a single model of robot the same set of instructions on a synchronized clock. Nothing new. Nothing amazing.）

“有点瘆人”派

《印度斯坦时报》收集了一些矛盾心理的评论：

“技术上很厉害，但也有点诡异。舞蹈本应是关于表达的，而这些机器人没有任何表情。”

“它们动作很好，但就像在看一场没有灵魂的传统表演。”

“为什么它们看起来这么瘆人？如果给它们装上类人的脸，会更可怕！”

技术解读：H1到底有多强

抛开情感争议，从纯技术角度看，这次表演确实是一个里程碑：

  • 16台机器人全自主协同：Unitree称这是世界上”首次大规模、全AI驱动、全自动化的人形机器人集群表演”
  • 最大关节扭矩360牛顿·米：这意味着强大的运动能力
  • 360°全景深度感知：精确的运动追踪
  • 3D激光SLAM导航：在复杂舞台环境中实现精准定位
  • AI实时音乐理解：机器人能根据音乐实时调整动作
  • 3个月AI驱动训练：每台机器人身高1.8米、重47公斤

这些不是遥控机器人，不是预编程播放器——它们是真正的AI驱动的自主系统。

更大的图景：春晚成为中国科技的”超级碗”

2025春晚的机器人表演并非孤立事件。就在几天前，DeepSeek AI刚刚在全球掀起风暴。两者叠加，让国际社会不得不重新审视中国的科技实力。

南华早报的后续报道透露，这次春晚上的成功让Unitree一炮而红，以至于2026年春晚已经有四家机器人公司（Unitree、Magiclab、Galbot、Noetix）与央视达成合作，据称合作金额高达约1亿元人民币。春晚，这个过去以歌舞小品相声为主的舞台，正在变成中国科技企业的”超级碗”。

对Unitree而言，春晚是一个完美的品牌引爆点。正如路透社所说，这个”不太可能的场所”，让人形机器人从实验室走进了主流视野。

冷思考：兴奋之后

海外的反应——无论是惊叹、焦虑还是质疑——其实都指向同一个事实：中国的机器人技术已经到了不容忽视的地步。

当然，也要看到理性声音的价值。Reddit用户说的没错，同步编舞和真正的通用人工智能之间还有很长的路要走。但作为一次公众展示，作为一个将”硬核科技”变成”文化事件”的营销案例，这场春晚机器人表演无疑是教科书级别的。

它告诉世界：中国不仅能造出先进的机器人，还能让它们扭秧歌、甩手绢——而且是在全球收视率最高的电视节目上。

这大概就是最好的科技自信。
